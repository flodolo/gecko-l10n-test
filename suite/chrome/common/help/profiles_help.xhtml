Gestion des profils

Si vous utilisez Internet chez vous et au travail, vous souhaiterez peut-être accéder à différents jeux de marque-pages, préférences, carnets d'adresses, comptes de messagerie, configuration de barre latérale, etc. De même, les différents membres de votre famille peuvent vouloir partager une copie du même navigateur, mais conserver leurs identités Internet séparées.

Le gestionnaire de profils permet de créer différents profils, chacun avec ses propres marque-pages, préférences, paramètres de messagerie, etc. Un profil par défaut est créé automatiquement lors de la première installation de votre logiciel de navigation. Après avoir créé un ou plusieurs profils supplémentaires, vous serez invité à préciser celui que vous souhaitez utiliser à chaque démarrage du navigateur.

Dans cette section :

  • Créer un nouveau profil
  • Supprimer ou renommer un profil

Créer un nouveau profil

Pour créer un profil :

  1. Ouvrez le gestionnaire de profils :
    Si &brandShortName; est ouvert
    Ouvrez le menu &tasksMenu.label; puis sélectionnez &switchProfileCmd.label;.
    Si &brandShortName; n'est pas lancé

    Sous Microsoft® Windows, ouvrez le menu Démarrer, choisissez Programmes, puis &brandShortName; et sélectionnez Gestionnaire de profils.

    Sous Mac OS, ouvrez le dossier de &brandShortName;, puis double-cliquez sur Gestionnaire de profils.

    Maintenez la touche Option enfoncée lorsque vous démarrez &brandShortName; à partir du Finder ou du Dock.

    Saisissez ce qui suit en ligne de commande : ./&brandShortName; -ProfileManager

    Sous Linux ou Unix, dans le répertoire d'installation de &brandShortName;, entrez la commande suivante dans un terminal : ./&brandShortName; -ProfileManager.

  2. Cliquez sur &manage.label;
  3. Cliquez sur Créer un profil…, lisez sa description, puis cliquez sur &goForwardCmd.label;.
  4. Entrez un nom de profil. Il peut s'agir de votre véritable nom, d'un surnom ou d'un nom associé à l'usage du profil, comme École.
  5. Acceptez le répertoire par défaut pour le nouveau profil, ou cliquez sur le bouton &button.choosefolder.label; pour choisir un autre répertoire.
  6. Cliquez sur Terminer.

Supprimer ou renommer un profil

Vous souhaitez supprimer des profils que vous n'utilisez plus. Pour supprimer ou renommer un profil :

  1. Ouvrez le gestionnaire de profils :
    Si &brandShortName; est ouvert
    Ouvrez le menu &tasksMenu.label; puis sélectionnez &switchProfileCmd.label;.
    Si &brandShortName; n'est pas lancé

    Sous Windows, ouvrez le menu Démarrer, choisissez Programmes, puis &brandShortName; et sélectionnez Gestionnaire de profils.

    Sous Mac OS, ouvrez le dossier de &brandShortName;, puis double-cliquez sur Gestionnaire de profils.

    Maintenez la touche Option enfoncée lorsque vous démarrez &brandShortName; à partir du Finder ou du Dock.

    Saisissez ce qui suit en ligne de commande : ./&brandShortName; -profilemanager

    Sous Linux ou Unix, dans le répertoire d'installation de &brandShortName;, entrez la commande suivante dans un terminal : ./&brandShortName; -profilemanager.

  2. Cliquez sur &manage.label;
  3. Pour supprimer un profil, sélectionnez son nom, puis cliquez sur Supprimer un profil…. Dans la boîte de confirmation, choisissez l'une des options suivantes :
    Supprimer les fichiers :
    Supprime tout le dossier de profil avec ses données stockées (marque-pages, préférences, comptes de courrier, etc. Assurez-vous de ne plus avoir besoin de quoi que ce soit dans le profil avant de choisir cette option.
    Ne pas supprimer les fichiers :
    Supprime le profil de la liste des profils disponibles mais conserve le répertoire du profil. En choisissant cette option, aucune des données stockées du profil ne sera effacée.
  4. Pour renommer un profil, sélectionnez son nom, cliquez sur Renommer un profil… et suivez les instructions.

Changer de profil

Si vous désirez changer de profil, vous pouvez le faire de deux façons différentes.

La première consiste à fermer &brandShortName;, à le relancer et à sélectionner un autre profil.

La seconde ne nécessite pas la fermeture de &brandShortName;.

  1. Ouvrez le menu &tasksMenu.label; puis sélectionnez &switchProfileCmd.label;.
  2. Choisissez le nouveau profil à utiliser.
  3. Cliquez sur &select.label;. &brandShortName; bascule directement vers le nouveau profil.

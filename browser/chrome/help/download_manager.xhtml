¿Qué es el administrador de descargas?

El administrador de descargas es una característica de &brandShortName; que permite seguir el rastro y controlar los archivos ya descargados y los actuales. Es una forma conveniente para escapar del desorden de las múltiples ventanas de descargas mientras se mantiene el acceso a las estadísticas de las descargas.

El administrador de descargas aparecerá cuando se descarguen archivos, al pulsar Herramientas > Descargas, y al pulsar las teclas &accelKey;+J&accelKey;+Y.

¿Qué puedo hacer con el administrador de descargas?

Aparte de mantener la información de todas las descargas en un único lugar, el administrador de descargas mantiene las acciones apropiadas de forma fácilmente accesible para cada descarga.

Pausa
Se puede poner en pausa cualquier descarga que está actualmente en progreso pulsando en el enlace de Pausa al lado de la entrada del archivo. Puede ser útil, por ejemplo, si se necesita realizar una pequeña descarga comenzada después de una descarga grande. Pausar las descargas le permite decidir qué descargas son las más importantes. Cuando se quiera continuar la descarga de estos archivos, simplemente use el enlace Continuar para finalizar estas descargas.
Cancelar
Si después de comenzar una descarga decide que ese archivo no es necesario, cancelar la descarga es sencillo: sólo hay que usar el enlace Cancelar junto a la información de la descarga para cancelarla y liberar la conexión para poder navegar y hacer otras descargas.
Abrir el archivo
Cuando ha finalizado una descarga, aparecerá junto a la entrada del archivo el enlace Abrir. Se usa para abrir el archivo.
Eliminar la entrada del archivo
Si no se desea guardar información de una descarga en particular, se puede eliminar la entrada con una sola pulsación del ratón. Sólo hay que usar el enlace Eliminar que está junto a la entrada del archivo para eliminarla.
Abrir carpeta contenedora
Si ha configurado &brandShortName; para guardar todas las descargas en una carpeta en las &pref.pluralCaps; del administrador de descargas, puede abrir esta carpeta haciendo clic en el nombre de la carpeta en la parte inferior de la ventana administrador de descargas.
Reintentar una descarga
Si por alguna razón la descarga no se completa, es muy fácil volver a intentarla. Para recuperar una descarga fallida y repetirla, solamente hay que usar el enlace Reintentar junto a la entrada del archivo y &brandShortName; reanudará la descarga.

&pref.pluralCaps; del administrador de descargas

Las &pref.plural; del administrador de descargas están accesibles abriendo la sección Descargas del panel Principal en la ventana de &pref.plural;. Le permiten cambiar varias &pref.plural; de descarga. Hay más información disponible sobre las diferentes &pref.plural; en la página de &pref.plural;.

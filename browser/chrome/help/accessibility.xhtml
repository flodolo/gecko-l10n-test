&brandShortName;內建許多輔助功能，可讓所有視力不佳、視覺障礙，或是行動不便的使用者更容易的瀏覽網頁。

使用鍵盤操作

鍵盤操作捷徑

通用快速鍵列表請見 &brandShortName; 快捷鍵。 若您已經連到網路，也可以查閱 &brandShortName;、 Microsoft Internet Explorer、以及 Opera 瀏覽器的 快捷鍵對照表。

總是使用鍵盤方向鍵瀏覽網頁

「鍵盤瀏覽功能」可讓您以游標遊走於網頁內容中，就像是在唯讀的記事本中操作一般。 您可以使用鍵盤選擇網頁，也能複製到剪貼簿中。要啟用此功能，請選擇 &pref.menuPath; ，開啟進階面板中的 一般分頁，並選取總是使用鍵盤方向鍵瀏覽網頁 &pref.singular;。

您也可以隨時按下 F7 切換此功能。當您按下 F7 時，&brandShortName; 會詢問 您是否要啟用此項功能，選取不要再顯示此視窗就可永遠關閉這個提示。

隨打即找

隨打即找可讓您快速尋找網頁中的文字或鏈結。這項功能提供兩種模式。 按下鍵盤上的 / 鍵後輸入您想於本頁搜尋的文字，或按鍵盤上的 ' 鍵搜尋本頁中的所有鏈結。&brandShortName; 會立刻將符合的文字 或鏈結以高亮度標記。

下面是本功能其他的快捷鍵：

  • 輸入更多字元可進一步縮小您的搜尋結果。&brandShortName; 會以高亮度標記出符合的鏈結或文字，或告訴您 找不到符合的內容。
  • 按 &accelKey;+G 或 F3 可跳到下一個符合的內容。
  • 按 &accelKey;+&shiftKey;+G 或 &shiftKey;+F3 可回到上一個符合的內容。
  • 按 Backspace 刪除前一個字元，&brandShortName; 也會自動還原位置至前一次找到的文字。
  • 按 Esc 取消搜尋。您也可在其他地方按一下、或稍等一下等搜尋列消失來取消搜尋。

您也可以改變此功能的預設模式，選擇 &pref.menuPath;，開啟進階面板中的一般分頁，再選 隨打即找 &pref.singular;。 如勾選此選項，您就可以隨時輸入要搜尋的文字進行搜尋，預設的動作會搜尋整個頁面的內容。您也可以按 ' 只搜尋鏈結或按 / 只搜尋文字。

這也可以找到並捲動到按鈕、文字方塊或其他表單元件的所在位置。只要先搜尋表單元件位置前的字詞，按下 Tab 即可。 再按一次 Tab 結束搜尋並移到高亮度標記後的下一個欄位或按鈕。

字體和色彩

使用自訂字型

有些網站使用難以閱讀的字型顯示文字，有些則用瀏覽器的預設字型顯示。您可以從 &pref.menuPath;， 選擇內容面板後，到字型與色彩設定您的預設字型與大小。

您可以按下進階鈕為預設字型設定不同樣式。由於許多網頁會自行指定字型， 您的設定有可能會被忽略。要強制使用您自己的字型，請不要勾選優先使用網頁指定的字型 &pref.singular;。

註：有些網頁可能無法正確的顯示您設定的字型。

設定最小字型大小

有些網站的字型設定過小以至無法舒適的閱讀。您可以設定最小字型大小讓這些頁面更容易閱讀。 &brandShortName; 會以您設定的最小大小為準顯示。

要設定最小字型大小，請至 &pref.menuPath;，選取內容面板，再按下 字型與色彩中的進階鈕。從下拉選單中設定最小字型大小。

註：有些網頁可能無法正確的套用您設定的最小字型大小。

縮放網頁文字

您可以從 檢視 > 文字大小 中暫時改變網頁的文字大小。 即使您瀏覽不同的網站，改變後的大小也會保留直到您關閉瀏覽器或該分頁時。

下面是其他有關文字大小的快捷鍵：

  • 按 &accelKey;++ 放大文字。範圍從原來大小的 120%, 150%, 200%, 300%, 450%, 675%, 1013%, 到 1519% 。
  • 按 &accelKey;+- 縮小文字。範圍從原來的 90%, 50%, 33%, 22%, 15%, 10%, 7%, 4%, 3%, 2%, 最小到 1%。
  • 按 &accelKey;+0 會回復到原本的文字大小。

註：有些網頁可能無法正確的縮放文字。

自訂頁面色彩

有些網站使用難以閱讀的文字或背景色彩顯示，有些則使用瀏覽器的預設色彩。 您可以從 &pref.menuPath; ，選擇 內容 面板後，並按下 字型與色彩中的色彩按鈕設定預設色彩。 您可以設定預設的文字與背景色彩、已拜訪或未拜訪的鏈結色彩，甚至選擇鏈結下方是否要加底線。

由於許多網頁會自行指定色彩，您的設定有可能會被忽略。要強制使用您自己的色彩設定， 請不要勾選優先使用網頁指定的字型 &pref.singular;。

使用高對比佈景主題

&brandShortName; 在您使用高對比佈景主題時，也會自動以高對比色彩顯示網頁及介面。 這會強制取代瀏覽器或網站的設定，&brandShortName; 的操作介面（選單、視窗、對話視窗等）也會隨之影響。

要使用高對比度佈景主題，請到 開始功能表 > 控制台 > 協助工具選項 > 顯示 ，選擇 使用高對比 &pref.singular;。

自訂網頁內容

阻擋彈出型視窗

請見 控制彈出型視窗 了解有關阻擋彈出型視窗的說明。

停用 Java Applets

有些網頁會用到 Java™ applets 提供豐富的互動體驗。但有些只能靠鍵盤瀏覽的 使用者卻會因為 Java applets 自動切換焦點卻沒有提供「跳離」該 Applet 的方式 而在瀏覽網路時遇到困難。如果這會對您造成困擾，您可以從 &pref.menuPath;， 選擇內容面板，不要勾選 可使用 Java &pref.singular; 取消對 Java 的支援。

限制 JavaScript 行為

您可以完全關閉 JavaScript 的支援。從 &pref.menuPath; 中， 選擇 內容面板，不要勾選可用 JavaScript 的選項即可。 但有些網頁會因為 JavaScript 被關閉而無法正確執行。

若您不希望完全關閉 JavaScript ，我們提供了一些有關 JavaScript 的設定限制其行為。 請按下可用 JavaScript 後的進階按鈕，打開 進階 JavaScript 設定視窗。您可以設定是否允許 scripts 移動或改變視窗的位置或大小、停用或取代右鍵選單、隱藏或改變狀態列的文字等。

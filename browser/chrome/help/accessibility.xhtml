Tillgänglighetsfunktioner i &brandFullName;

&brandShortName; har flera funktioner som gör att webbläsaren och innehåll på webbsidor är tillgängliga för alla användare, även för de med svag syn eller blinda, eller de som har svårt att använda tangentbord eller mus.

Använda tangentbordet

Kortkommandon

En lista på de vanligaste kortkommandona finns i avsnittet &brandShortName; kortkommandon. På Internet finns en jämförelselista över kortkommandon i &brandShortName;, Microsoft Internet Explorer och Opera.

Använd alltid piltangenterna för att navigera i sidor

Använd alltid piltangenterna för att navigera i sidor låter dig hoppa mellan webbinnehåll som om du arbetade i en redigerare. Du kan använda tangentbordet för att markera innehåll på en webbsida och kopiera det till urklipp. Du kan aktivera denna funktion genom att välja &pref.menuPath;, öppna fliken Allmänt i panelen Avancerat och kryssa i Använd alltid piltangenterna för att navigera i sidor.

Du kan även trycka F7 för starta eller stänga av denna funktion. När du trycker F7 kommer &brandShortName; att fråga om du verkligen vill använda funktionen. Du kan avaktivera denna fråga genom att välja Visa inte denna dialogruta igen.

Sök efter text när jag börjar skriva

Sök efter text när jag börjar skriva låter dig snabbt gå till text eller hyperlänkar på en webbsida. Denna funktion har två lägen, tryck / och börja skriv för att söka bland all text på den aktuella sidan, eller tryck ' innan du skriver för att endast söka bland hyperlänkar. &brandShortName; kommer att markera matchande text eller länk under tiden du skriver.

Följande kortkommandon styr denna funktion:

  • Skriv in fler tecken för att begränsa sökningen. &brandShortName; kommer att markera nästa matchande länk eller text, eller meddela att inget hittades.
  • Tryck &accelKey;+G eller F3 för att hitta nästa förekomst av den aktuella söksträngen.
  • Tryck &accelKey;+&shiftKey;+G eller &shiftKey;+F3 för att hitta föregående förekomst av den aktuella söksträngen.
  • Tryck Backsteg för att ångra det senaste tecknet du skrev. &brandShortName; återgår till den senast markerade texten.
  • Tryck Esc för att avbryta en sökning. Du kan även avbryta en sökning genom att byta fokus, rulla sidan med musen eller tangentbordet, eller helt enkelt vänta tills tiden löper ut.

Du kan ändra standardbeteendet för denna funktion genom att välja &pref.menuPath;, öppna fliken Allmänt i panelen Avancerat och kryssa i Sök efter text när jag börjar skriva. Om detta alternativ är valt startar en sökning automatiskt när du börjar skriva. En sådan automatisk sökning söker normalt bland all text på sidan. Du kan fortfarande trycka ' eller / för att välja sökmetod.

Du kan använda denna funktion för att gå till knappar, textrutor och andra formulärkontroller. Sök på texten strax före formulärkontrollen du vill gå till och tryck sedan på Tabb. När du trycker på Tabb avslutas automatiskt sökningen och fokus flyttas till nästa formulärfält eller knapp efter den markerade texten.

Teckensnitt och färger

Åsidosätta sidans teckensnitt

En del webbsidor använder teckensnitt som kan vara svåra att läsa, och andra sidor kan vara beroende av webbläsarens standardteckensnitt. Du kan välja standardteckensnitt genom att gå till &pref.menuPath;, välja panelen Innehåll och sedan välja ett teckensnitt och en storlek i sektionen Teckensnitt & Färger.

Du kan ange standardteckensnitt för olika stilar, inklusive serif, sans-serif och fast bredd genom att klicka på knappen Avancerat…. Många webbsidor anger dock egna teckensnitt och dina val kommer inte att ha någon effekt på sådana sidor. För att åsidosätta teckensnitt angivna på sådana sidor, avmarkera Låt sidor använda egna teckensnitt, i stället för ovanstående.

Tänk på att vissa webbsidor kanske inte visas på rätt sätt beroende på vilket teckensnitt du valt.

Ange en minimistorlek

En del webbplatser kanske har text som är för liten för att du ska kunna läsa den. För att underlätta läsning av sådana sidor kan du ange en minimistorlek på text. På webbsidor som försöker visa en text med mindre storlek än denna kommer &brandShortName; att förstora texten till den angivna minimistorleken.

För att ange en minimistorlek på text, gå till &pref.menuPath;, välj panelen Innehåll och klicka på knappen Avancerat… i sektionen Teckensnitt & Färger. Du kan välja en minimistorlek från rullgardinsmenyn Minsta storlek:.

Tänk på att vissa webbsidor kanske inte visas på rätt sätt beroende på ditt val av minimistorlek.

Ändra storlek på sidtext

Du kan temporärt ändra storlek på texten på en webbsida i menyn Visa > Textstorlek. Den nya storleken kommer då att användas tills du stänger fönstret eller fliken, även om du går till en annan webbplats.

Följande kortkommandon styr textstorleken:

  • Tryck &accelKey;++ för att öka textstorleken. Detta zoomar texten till 120% av den ursprungliga storleken, sedan till 150%, 200%, 300%, 450%, 675%, 1013% och 1519%.
  • Tryck &accelKey;+- för att minska textstorleken. Detta zoomar texten till 90% av den ursprungliga storleken, sedan till 50%, 33%, 22%, 15%, 10%, 7%, 4%, 3%, 2% och 1%.
  • Tryck &accelKey;+0 för att återställa textstorleken till webbsidans standardstorlek.

Tänk på att vissa webbsidor kanske inte visas på rätt sätt om du ökar eller minskar textstorleken.

Åsidosätta sidfärger

En del webbplatser kan använda färger för text och bakgrund som gör det svårt att läsa sidan medan andra platser använder webbläsarens standardfärger. Du kan ange standardfärger genom att gå till &pref.menuPath;, välja panelen Innehåll och klicka på knappen Färger… i sektionen Teckensnitt & Färger. Där kan du välja färger för text och bakgrund och även vilka färger som ska användas för besökta och obesökta hyperlänkar. Du kan även välja om hyperlänkar ska vara understrukna.

Många webbsidor anger dock egna färger så standardfärgerna kommer inte alltid att användas. För att åsidosätta färger angivna av webbsidor, avmarkera Låt sidor använda egna färger, i stället för ovanstående.

Använda ett högkontrasttema

&brandShortName; upptäcker automatiskt om du använder ett högkontrasttema och visar allting med detta färgschema. Alla andra inställningar för webbläsaren eller webbsidor åsidosätts och gränssnittet i &brandShortName; (alla menyer, fönster och dialogrutor) och innehållet på webbsidor anpassas.

För att använda ett tema med hög kontrast, gå till Start-menyn > Kontrollpanelen > Hjälpmedel > Bildskärm och markera alternativet Använd Högkontrast.

Styra webbinnehåll

Se Kontrollera popup-fönster för information om hur du blockerar popup-fönster.

Stänga av Java-appletar

En del webbplatser använder Java™-appletar för att ge ett rikare innehåll till sina sidor. För de som navigerar med tangentbordet kan det leda till problem med vissa Java-appletar som automatiskt sätter fokus och inte ger någon möjlighet till att ”gå ur” appleten för att navigera till andra delar av webbsidan. Om detta är ett problem för dig kan du avaktivera Java genom att gå till &pref.menuPath;, välja panelen Innehåll och avmarkera Aktivera Java.

Begränsa JavaScript-funktioner

Du kan stänga av JavaScript-användning helt genom att gå till &pref.menuPath;, välja panelen Innehåll och avmarkera Aktivera JavaScript. En del webbsidor är dock beroende av JavaScript och kanske inte fungerar på rätt sätt om du avaktiverar JavaScript.

Om du inte vill avaktivera JavaScript helt finns det andra inställningar du kan göra som begränsar användningen av JavaScript. Till höger om kryssrutan Aktivera JavaScript kan du klicka på knappen Avancerat… för att öppna fönstret Avancerade JavaScript-funktioner. Där kan du bestämma om skript ska kunna flytta eller ändra storlek på fönster, höja eller sänka fönster, avaktivera eller byta ut snabbmenyer, dölja Status-fältet eller ändra texten i Status-fältet.

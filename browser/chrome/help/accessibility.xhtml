Características de accesibilidad de &brandFullName;

&brandShortName; incluye muchas características para hacer el navegador y el contenido web accesibles a todos los usuarios, incluyendo aquéllos que tienen problemas de visión (pérdida total o parcial), o una capacidad limitada para usar un teclado o ratón.

Usar un teclado

Atajos de teclado

Hay disponible una lista de atajos de teclado comunes en Atajos de teclado de &brandShortName;. Puede encontrar en línea una tabla comparativa de atajos de teclado implementados en &brandShortName;, Microsoft Internet Explorer y Opera.

Usar siempre los cursores del teclado para navegar por las páginas

Usar siempre las teclas del cursor para navegar dentro de las páginas le permite moverse a través del contenido web como si estuviera en el interior de un editor de sólo lectura. Puede usar el teclado para seleccionar el contenido web y copiarlo al portapapeles. Puede activar esta característica seleccionando &pref.menuPath;, abriendo la pestaña General en el panel Avanzado, y marcar la &pref.singular; Usar siempre las teclas del cursor para navegar dentro de las páginas.

Puede pulsar F7 en cualquier momento para activar y desactivar esta característica. Cuando pulse F7, &brandShortName; le preguntará si realmente quiere activar esta característica. Puede desactivar esta pregunta seleccionando No volver a mostrar este diálogo.

Buscar texto mientras se escribe

Buscar texto mientras se escribe le permite navegar rápidamente por el texto o hiperenlaces de una página web. Esta característica tiene dos modos. Pulse / y comience a escibir para buscar en todo el texto de la página actual, o pulse ' seguido de la cadena a buscar para hacerlo únicamente en los hiperenlaces. &brandShortName; resaltará el texto o hiperenlaces que coincidan con su búsqueda a medida que escriba.

Los siguientes atajos de teclado controlan esta característica:

  • Escriba más caracteres para refinar su búsqueda. &brandShortName; resaltará el siguiente texto o hiperenlace coincidente o le alertará si no hay ninguna coincidencia.
  • Pulse &accelKey;+G o F3 para encontrar la próxima ocurrencia de su cadena de búsqueda actual.
  • Pulse &accelKey;+&shiftKey;+G o &shiftKey;+F3 para encontrar la ocurrencia anterior de su cadena de búsqueda actual.
  • Pulse Borrar para eliminar el último carácter que ha escrito. &brandShortName; revertirá al texto previamente resaltado.
  • Pulse Esc para cancelar una búsqueda. También puede cancelar una búsqueda cambiando el foco, desplazando la página con el ratón o el teclado, o simplemente esperando hasta alcanzar un tiempo de caducidad.

Puede cambiar el comportamiento predeterminado de esta característica seleccionando &pref.menuPath;, abriendo la pestaña General del panel Avanzado y marcando la &pref.singular; Buscar texto mientras se escribe. Si está marcada esta &pref.singular;, puede comenzar a escribir en cualquier momento para iniciar automáticamente una búsqueda en el texto. Por omisión, las búsquedas iniciadas automáticamente examinan todo el texto en la página. Aún así, puede pulsar explícitamente ' o / para comenzar a buscar en un modo o el otro.

Puede usar esta característica para acceder a los botones, cajas de texto y otros controles de formularios. Busque el texto que precede inmediatamente al control de formulario que busca y pulse Tab. Pulsar Tab termina automáticamente la búsqueda y fija el foco al siguiente campo o botón de formulario tras el texto resaltado.

Tipografía y colores

Reemplazar tipografía de la página

Algunos sitios web pueden mostrar texto con tipografías que resultan difíciles de leer, y otros sitios pueden basarse en los tipos de letra por omisión del navegador. Puede establecer sus tipos de letra predeterminados acudiendo a &pref.menuPath;, seleccionando el panel Contenido, y seleccionando un tipo y tamaño de letra en la sección Tipografía y colores.

Puede especificar tipos de letra por omisión para diferentes estilos, incluyendo serif, sans-serif y monoespaciado, pulsando el botón Avanzadas. Sin embargo, muchas páginas web especifican sus propios tipos de letra, así que su elección no tendrá efecto en estas páginas. Para reemplazar las tipografías definidas por tales páginas, desmarque la &pref.singular; Permitir a las páginas elegir sus propios tipos de letra, en vez de mi selección anterior.

Observe que algunas páginas web pueden mostrarse de manera incorrecta dependiendo de qué tipografía por omisión haya elegido usted.

Fijar un tamaño mínimo de letra

Algunos sitios web pueden mostrar texto que es demasiado pequeño para leerlo cómodamente. Para hacer estas páginas más fáciles de leer, puede establecer un tamaño de letra mínimo. En las páginas web que intenten mostrar texto menor que este tamaño, &brandShortName; agrandará el texto al tamaño escogido por usted.

Para fijar un tamaño de letra mínimo, vaya a &pref.menuPath;, seleccione el panel Contenido, y haga clic en el botón Avanzadas en la sección Tipografía y colores. Puede especificar un tamaño mínimo de letra del menú desplegable.

Observe que algunas páginas web pueden mostrarse de manera incorrecta dependiendo del tamaño mínimo de letra que haya especificado usted.

Redimensionar el texto de las páginas

Puede cambiar temporalmente el tamaño del texto en cualquier página web en el menú Ver > Tamaño del texto. El nuevo tamaño tendrá efecto hasta que cierre la pestaña o ventana del navegador, incluso si navega a un sitio web diferente.

Los siguientes atajos de teclado controlan el tamaño del texto:

  • Pulse &accelKey;++ para incrementar el tamaño del texto. Esto agranda el texto al 120% de su tamaño original, luego el 150%, 200%, 300%, 450%, 675%, 1013% y 1519%.
  • Pulse &accelKey;+- para decrementar el tamaño del texto. Esto reduce el texto al 90% de su tamaño original, luego el 50%, 33%, 22%, 15%, 10%, 7%, 4%, 3%, 2% y 1%.
  • Pulse &accelKey;+0 para restablecer el tamaño del texto al indicado originalmente por la página web.

Observe que algunas páginas web pueden mostrarse de manera incorrecta si incrementa o decrementa el tamaño de letra.

Reemplazar los colores de la página

Algunos sitios web pueden mostrar colores de texto y fondo que son difíciles de leer, y otros pueden basarse en los colores por omisión del navegador. Puede fijar sus colores predeterminados si va a &pref.menuPath;, seleccionando el panel Contenido y haciendo clic en el botón Colores de la sección Tipografía y colores. Puede indicar colores predeterminados para texto y fondo, así como para hiperenlaces visitados y no visitados. También puede especificar si los hiperenlaces deben ser subrayados.

Sin embargo, muchas páginas web establecen sus propios colores, así que las opciones indicadas por usted no tendrán efecto. Para reemplazar los colores definidos por las páginas web, desmarque la &pref.singular; Permitir a las páginas elegir sus propios colores, en vez de mi selección anterior..

Usar un tema de alto contraste

&brandShortName; detecta automáticamente si está usando un tema de alto contraste y muestra todo en esquema de color de alto contraste. Esto prevalece sobre cualquier otra configuración del navegador o de las páginas web, y afecta al interfaz de &brandShortName; en sí mismo (todos los menús, ventanas y cajas de diálogo) y al contenido de cualquier página web que visite.

Para usar un tema de alto contraste, vaya a Menú Inicio > Panel de control > Opciones de accesibilidad > Pantalla y marque la &pref.singular; Usar alto contraste.

Controlar el contenido web

Bloquear ventanas emergentes

Vea Controlar ventanas emergentes para información sobre cómo bloquear ventanas emergentes.

Desactivar applets de Java

Algunas páginas proporcionan una rica experiencia interactiva con applets de Java™. Sin embargo, algunos usuarios que se apoyan en la navegación con teclado pueden experimentar problemas con algunos applets de Java que obtienen automáticamente el foco y no proporcionan una manera de salir del applet y navegar al resto de la página web. Si esto es un problema para usted, puede desactivar Java si va a &pref.menuPath;, seleccionando Contenido y desmarcando la &pref.singular; Activar Java.

Limitar el comportamiento de JavaScript

Puede desactivar JavaScript por completo si va a &pref.menuPath;, seleccionando el panel Contenido y desmarcando Activar JavaScript. Algunas páginas web se apoyan en JavaScript y pueden no funcionar correctamente si JavaScript está desactivado.

Si no quiere desactivar JavaScript por completo, también hay un cierto de preferencias avanzadas de JavaScript para restringir el comportamiento de los scripts. Inmediatamente tras la casilla Activar JavaScript, pulse el botón Avanzado para abrir la ventana Parámetros avanzados de JavaScript. Puede controlar si se permite a los scripts mover o redimensionar ventanas existentes, traerlas a primer plano o llevarlas fondo, desactivar o reemplazar menús contextuales, ocultar la barra de estado, o cambiar el texto de la barra de estado.

&brandFullName; 易用性

&brandShortName; 包含了许多特性，可以使得浏览器和网页内容对所有用户都更加易用，包括视力低下的人甚至盲人，或者行动不便，无法使用键盘或者鼠标的人。

使用键盘

键盘快捷键

常用键盘快捷键列表位于 &brandShortName; 键盘快捷键。 如果在线, 您可以找到一个快捷键对比图表，其中包括&brandShortName;、 Microsoft Internet Explorer，以及 Opera。

使用键盘选择文本

使用键盘选择文本特性可以让您在只读编辑器中移动。您可以使用键盘来选择网页内容并将其复制到剪贴板。您可以通过以下方式启用此特性：选择 &pref.menuPath;, 打开高级面板中的常规标签页，然后选中允许使用键盘选择文本 &pref.singular;。

您可以使用 F7 键在任何时候打开或关闭此特性。当您按 F7 时, &brandShortName; 将询问是否启用此特性，您也可以通过选择不再显示此对话框来关闭提示。

输入文字时同步查找

输入文字时同步查找允许您快速定位至网页中的文本或者超级链接。此特性有两种模式。按 / 然后开始输入，即可查找当前页面的所有文本；按 ' 则开始查找超级链接。&brandShortName; 将自动加亮显示您输入的文本或者链接。

下列键盘快捷键用于控制此特性：

  • 输入更多字符可精确搜索。&brandShortName; 会加亮显示下一个匹配的链接或者文本，或者发出警告音提示无法匹配。
  • 按 &accelKey;+G 或 F3 来查找下一个符合搜索条件的字符串。
  • 按 &accelKey;+&shiftKey;+G 或者 &shiftKey;+F3 来查找上一个符合搜索条件的字符串。
  • 按 退格 撤销最后输入的一个字符。&brandShortName; 将回到上一个加亮的文本。
  • 按 Esc 来取消搜索。您也可以通过变换焦点来取消搜索，使用鼠标或键盘来滚动页面，或者干脆等待超时。

您可以通过选择 &pref.menuPath;, 打开高级面板中的常规标签，然后选中输入时即时查找 &pref.singular;。如果此选项被选中，您可以在任何时候输入文字并自动开始进行内联查找。默认情况下，自动开始的查找会搜索页面中的所有文字。您也可以按 ' 或 / 来选择一种模式进行查找。

您可以使用此特性来快速切转至按钮、文本框或者其它表单控件。搜索您想要转至的控件之前的文本并按 Tab 键。按 Tab 键自动退出搜索并将焦点转至下一个高亮文本后的表单域或按钮。

字体和颜色

忽略页面字体

某些网站可能会用难以阅读的字体来显示文本，其它一些站点也可能依赖于浏览器的默认字体。您可以通过以下方式设置默认字体：转到 &pref.menuPath;，选择内容面板，然后在字体 & 颜色区域选择一个默认字体和大小。

只需要单击高级按钮，您就可以为不同风格的字体，包括衬线字体、无衬线字体，以及等宽字体指定默认字体。但是，因为许多网页都会自己指定字体，因此您的选择并不会对那些页面起作用。要忽略页面的字体设置，取消选中允许页面选择显示字体而无需使用上面的设置 &pref.singular;即可。

请注意在您的默认字体设置下，某些网页可能会显示的不正常。

设置最小字体

某些网站可能会使用非常小的文本来显示，以至于影响到了阅读。您可以设置一个最小的字体来提高这类网页的可读性。当遇到所用字体比这个字体更小的网页时，&brandShortName; 会将字体放大到您指定的大小。

要设置最小字体，您可以通过选择 &pref.menuPath;，选择内容面板，然后单击字体 & 颜色区域中的高级按钮，并在下拉菜单中指定最小字体。

请注意在您的最小字体设置下，某些网页可能会显示的不正常。

文本缩放

通过“查看“>“文字大小“菜单，您可以临时改变任何网页的文本大小。新尺寸将持续有效（即使您浏览到一个不同的网站），直到您关闭浏览器窗口或标签页。

下列键盘快捷键用于控制文本大小：

  • 按 &accelKey;++ 增加文字大小。这将文字增大到原始尺寸的 120%，然后 150%, 200%, 300%, 450%, 675%, 1013%, 以及 1519%。
  • 按 &accelKey;+- 减小字大小。这将文字减小到原始尺寸的 90%，然后 50%, 33%, 22%, 15%, 10%, 7%, 4%, 3%, 2%, 以及 1%。
  • 按 &accelKey;+0 将文字大小重置为页面默认值。

请注意，您增大或缩小字体后，某些网页可能会显示的不正常。

忽略页面颜色

某些网站可能会使用一些难以阅读的颜色来显示文本或者背景色。而其它一些网站可能会依赖浏览器的默认颜色。您可以通过以下方式设置默认颜色：转至 &pref.menuPath;，选择内容面板，然后在字体 & 颜色区域单击颜色按钮。您可以指定默认的文字或背景色，以及访问或未访问过的超级链接的默认颜色。您还可以在此指定是否给超级链接加下划线。

但是，因为许多网站都会自己指定颜色，因此您的设置可能会无效。要忽略网页的颜色设置，您需要取消选中允许页面选择显示颜色而无需使用上面的设置&pref.singular;。

使用高对比度主题

&brandShortName; 自动探测您是否正在使用高对比度主题。如果是的话，它将自动使用您的主题来显示一切。这会忽略一切其它浏览器或网页的设置，而且还会影响 &brandShortName; 的界面本身（所有菜单、窗口以及对话框）以及任何您访问到的网页内容。

要使用高对比度主题，转到“启动菜单”>“控制面板”>“易用性选项”>“显示”，然后选中使用高对比主题&pref.singular;。

控制网页内容

阻止弹出窗口

参见控制弹出窗口得到更多关于弹出窗口的信息。

关闭 Java 小应用

某些网站提供支持丰富交互体验的 Java™ 小应用。尽管如此，某些依赖键盘进行浏览的用户可能会因此遇到因为某些 Java 小应用自动设置焦点并且不允许跳出该应用以便浏览网页的其它部分的情况。如果您遇到了这个问题，您可以通过以下方式禁用 Java：转至 &pref.menuPath;，选择内容面板，然后取消选中启用 Java&pref.singular;。

限制 Javascript 行为

您可以完全关闭 JavaScript，只需要转至 &pref.menuPath;，选择内容面板，然后取消选中启用 JavaScript。某些依赖 JavaScript 的网站可能会因此无法正常工作。

如果您不想完全禁用 JavaScript，您也可以通过高级 JavaScript 设置来限制脚本的行为。就在启用 JavaScript复选框之后，单击高级按钮即可打开高级 JavaScript 设置窗口。您可以控制是否允许脚本移动或修改窗口大小、抬升或降低窗口、启用或替换上下文菜单、隐藏状态栏，或者修改状态栏文本。

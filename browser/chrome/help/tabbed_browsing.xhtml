Navigarea cu file

Navigarea cu file vă permite să deschideți file pentru fiecare pagină web deschisă, într-o singură fereastră &brandShortName;. Puteți deschide legături noi în file noi, unde ele se vor încărca în timp ce dumneavoastră citiți pagina curentă. Nu trebuie să folosiți navigarea cu file, dar dacă o folosiți, s-ar putea să găsiți navigarea cu file o metodă mai bună și mai rapidă de a naviga pe web.

În această secțiune:
  • Crearea și încărcarea filelor
  • Mutarea filelor într-o fereastră
  • Închiderea filelor
  • Personalizarea navigării cu file
  • Ponturi și trucuturi

Crearea și încărcarea filelor

Pentru a deschide o filă nouă, apăsați &accelKey;+T, selectați Fișier > Filă nouă, sau dublu clic pe un spațiu gol în bara cu capetele filelor.

Pentru a încărca o legătură într-o filă, puteți proceda astfel:

  • Clic mijlociu pe legătură. (Dacă mausul dumneavoastră are o rotiță, apăsatul ei este echivalent cu clic mijlociu.)
  • Trageți legătura și plasați-o pe un spațiu gol din bara cu capetele filelor. (Dacă este deschisă doar o pagină, bara cu capetele filelor ar putea fi ascunsă. Vedeți Personalizarea navigării cu file pentru mai multe informații despre cum puteți schimba această funcționalitate.)
  • Trageți și plasați legătura pe un cap de filă pentru a deschide legătura în acea filă.
  • Clic dreaptaApăsați &ctrlKey;, clic pe legătură și alegețiDeschide legătura în filă nouă din meniul contextual.

Pentru a deschide o adresă din bara adresei într-o filă nouă, introduceți adresa și apăsați &altKey;+&enterKey;.

Mutarea filelor într-o fereastră

Filele sunt afișate în ordinea în care au fost deschise, ordine care ar putea să nu fie ceea ce doriți dumneavoastră. Pentru a muta o filă în alt loc dintr-o fereastră &brandShortName; trageți-o și plasați-o unde o doriți folosind mausul dumneavoastră. În timp ce trageți fila, &brandShortName; arată un mic indicator pentru a vă arăta unde va fi mutată fila. Alternativ, dacă doriți, puteți folosi scurtături de taste pentru a muta filele într-o fereastră.

Închiderea și restaurarea filelor închise

Pentru a închide fila curentă, puteți să apăsați &accelKey;+W, să clicați pe butonul Închide fila clic mijlociu pe filă sau să selectați Fișier > Închide fila. Pentru a închide toate filele în afară de cea curentă, clic dreaptaapăsați &ctrlKey;, clic pe filă și alegeți Închide celelalte file.

Ferestrele &brandShortName; păstrează o listă a filelor pe care le-ați închis recent. Puteți restaura orice filă selectând-o din Istoric > File închise recent. Puteți restaura toate filele selectând Istoric > File închise recent > Deschide toate în file . Apăsați &accelKey;+&shiftKey;+T pentru a le deschide una câte una în ordine inversă.

Personalizarea navigării cu file

Pentru a schimba &pref.plural;le navigării cu file, selectați &pref.menuPath; și deschideți panoul File.

Ponturi și trucuri

Semnele de carte și navigarea cu file
Pentru a pune semn de carte la un set de file:
  1. Apăsați &accelKey;+&shiftKey;+D sau selectațiSelectați Semne de carte > Semn de carte la toate filele pentru a adăuga toate filele din fereastra curentă la un nou dosar cu semne de carte.
  2. Pentru a deschide semnele de carte în file, deschideți un dosar cu semne de carte și alegeți Deschide în file. Puteți de asemenea să apăsați butonul mijlociu al mausului pe dosar și să deschideți conținutul său în file. Semnele de carte se vor deschide în file existente deja, nu în file noi.
File ca pagină de start
În loc să folosiți doar o pagină web ca pagina dumneavoastră de start, puteți seta ca pagină de start un set de file. Selectați saiturile dumneavoastră favorite și deschideți-le în file într-o singură fereastră. Selectați &pref.menuPath; și deschideți panoul General. Sub Pornire, apăsați butonul Folosește paginile curente. De acum, când apăsați butonul Acasă, saiturile dumneavoastră favorite se vor încărca cu un singur clic.
Adăugarea unui buton pentru deschiderea unei file noi
Puteți adăuga un buton Filă nouă pe bara de unelte pentru a deschide ușor file noi personalizând barele de unelte.
Scurtături de taste
&brandShortName; are o listă lungă de scurtături de taste pentru navigarea cu file; vedeți lista întreagă în Scurtături de taste.

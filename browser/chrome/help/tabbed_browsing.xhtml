Navegación con pestañas

La navegación con pestañas le permite abrir pestañas, cada una de ellas mostrando una página web, dentro de una única ventana de &brandShortName;. Puede abrir enlaces en nuevas pestañas, donde se cargarán mientras usted lee la página actual. No tiene que usar la navegación con pestañas, pero si lo hace descubrirá que es una manera mejor y más rápida de navegar por la web.

Crear y cargar pestañas

Para abrir una nueva pestaña, pulse &accelKey;+T, seleccione Archivo > Nueva pestaña, o haga doble clic en un espacio vacío de la barra de pestañas.

Para cargar un enlace en una pestaña, puede hacer una de las siguientes cosas:

  • Hacer clic con el botón central en el enlace (si tiene un ratón con rueda, hacer clic con la rueda tiene el mismo efecto).
  • Arrastrar el enlace y soltarlo en un espacio vacío de la barra de pestañas (si sólo hay una página web abierta, la barra de pestañas puede estar oculta. Vea Personalizar la navegación con pestañas para información sobre cómo cambiar esto).
  • Arrastre y suelte el enlace en una pestaña existente para abrir el enlace en esa pestaña.
  • Haga clic derechoPulse &ctrlKey;, y haga clic en el enlace y elija Abrir enlace en nueva pestaña del menú contextual.

Para abrir una URL en la barra de direcciones en una nueva pestaña, introduzca la URL y pulse &altKey;+&enterKey;.

Mover pestañas dentro de una ventana

Las pestañas se muestran en el orden en que las abre, lo que puede no ser siempre lo que se quiere. Para mover una pestaña a una posición diferente dentro de una ventana de &brandShortName;, simplemente arrástrela usando el ratón. Mientras arrastra la ventana, &brandShortName; muestra un pequeño indicador para señalar dónde se moverá la pestaña. Alternativamente, puede usar atajos de teclado para mover pestañas dentro de una ventana si lo desea.

Cerrar y recuperar pestañas

Para cerrar la pestaña actual, pulse &accelKey;+W, haga clic en el botón Cerrar pestaña, haga clic con el botón central en la pestaña, o seleccione Archivo > Cerrar pestaña. Para cerrar todas las pestañas distintas de la actual, haga clic con el botón derechopulse &ctrlKey;, y haga clic en la pestaña y elija Cerrar las otras pestañas.

&brandShortName; mantiene una lista de pestañas que se han cerrado recientemente. Puede recuperar cualquier pestaña selecionándola en Historial > Pestañas cerradas recientemente. Puede recuperar todas las pestañas seleccionando Historial > Pestañas cerradas recientemente > Abrir todas en pestañas . Presione &accelKey;+&shiftKey;+T para abrirlas una a una en orden inverso.

Personalizar la navegación con pestañas

Para cambiar las &pref.plural; de navegación con pestañas, seleccione &pref.menuPath; y abra el panel Pestañas.

Consejos y trucos

Marcadores y navegación con pestañas
Para añadir a marcadores un conjunto de pestañas:
  1. Pulse &accelKey;+&shiftKey;+D o seleccioneSeleccione Marcadores > Añadir todas las pestañas a marcadores.
  2. Para abrir marcadores en pestañas, abra una carpeta de marcadores y elija Abrir en pestañas. También puede hacer clic con el botón central en la carpeta para abrir su contenido en pestañas. Los marcadores se abrirán en pestañas existentes, no en pestañas nuevas.
Pestañas como página de inicio
En lugar de usar sólo una página web como su página de inicio, puede hacer que ésta sea un conjunto de pestañas. Seleccione sus sitios web favoritos y ábralos en pestañas en una sola ventana. Seleccione &pref.menuPath; y abra el panel Principal. Bajo Inicio, pulse el botón Usar páginas actuales. A partir de ahora, cuando pulse el botón Inicio se cargarán sus sitios web favoritos con un solo clic.
Añadir un botón Nueva pestaña a la barra de herramientas
Puede añadir un botón Nueva pestaña en la barra de herramientas para abrir fácilmente nuevas pestañas personalizando sus barras de herramientas.
Atajos de teclado
&brandShortName; tiene un amplio conjunto de atajos de teclado relacionados con la navegación con pestañas; vea la lista completa en Atajos de teclado.

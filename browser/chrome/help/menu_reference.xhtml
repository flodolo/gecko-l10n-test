&brandShortName;

关于 &brandFullName;

显示一个对话框，其中显示关于 &brandShortName; 的一些信息， 包括当前版本以及一个简单的致谢名单。

首选项...

显示&pref.pluralCaps; 窗口, 在这里您可以修改 &brandShortName; 里的许多 &pref.plural;。

退出 &brandShortName;

关闭 &brandShortName; 中所有打开的窗口，包括正在进行的下载以及其他 &brandShortName; 窗口。如果在您选择这一菜单项时，还有其他打开的标签页或窗口，&brandShortName; 会询问您以防止误操作。您可以通过以下方式关闭该警告：在 &pref.menuPath; 的标签式浏览面板中，取消选中关闭多个标签页时警告我复选框。如果您正在下载任何文件，&brandShortName; 也会警告你。

文件

新建窗口

打开一个新的 &brandShortName; 窗口。

新建标签页

在当前 &brandShortName; 窗口中打开一个新的标签页。

打开位置...

光标会定位到 &brandShortName; 的地址栏上，您可以输入一个新地址来访问它。

打开文件...

显示一个“打开文件”对话框，在这里您可以从本地硬盘或网络上选择文件。 您可以在 &brandShortName; 中查看多种类型的文件，包括 HTML/XML、图像文件、文本文件等。

关闭（窗口）

关闭当前窗口。

关闭标签页

关闭当前标签页并选中最右边的标签页。 本菜单项仅在当前窗口中打开的标签页多于一个时才可见。

页面另存为...

保存您正在浏览的网页。您可以选择保存包括图片在内的整个网页、 HTML 页面内容或文本页面内容。

发送链接...

启动您的计算机上的缺省电子邮件程序，打开一个写邮件的窗口， 然后就可以发出一个包含当前页面链接的邮件。

页面设置...

显示一个页面设置对话框，在这里您可以设定一些打印选项，比如页边距、 页眉、页脚以及页面方向等。

打印预览

显示当前文档打印后的效果。它可以反映出页面设置窗口中的设置 以及页面作者的一些设置。

打印...

显示打印对话框，在这里您可以设定打印份数等参数，单击打印即可打印当前页面。

导入...

打开“导入向导”对话框，在这里您可以从其他浏览器导入 &pref.plural;、书签、历史、密码以及其他数据，这些浏览器包括微软 Internet Explorer、Netscape、Mozilla 或 Opera。

脱机工作

在脱机和在线模式之间切换。脱机模式可以让您在不连接到互联网的情况下 查看您以前访问过的网页。

退出 退出

关闭 &brandShortName; 中所有打开的窗口，包括正在进行的下载以及其他 &brandShortName; 窗口。如果在您选择这一菜单项时，还有其他打开的标签页或窗口，&brandShortName; 会询问您以防止误操作。您可以通过以下方式关闭该警告：在 &pref.menuPath; 的标签式浏览面板中，取消选中关闭多个标签页时警告我复选框。如果你正在下载任何文件，&brandShortName; 也会警告你。

编辑

撤消

撤消在文本域内的最后一次操作。如果您稍后又不想撤消了， 使用“重做”命令即可。

重做

重新执行上一次撤消的命令。

剪切

从文本域中剪切选中的文本并存到剪贴板中。使用“粘贴”命令即可将它粘贴到其他地方。

复制

把选中的文字拷贝到剪贴板中。使用“粘贴”命令可将它粘贴到文本域中。

粘贴

把存在剪贴板中的文字粘贴到文本域中。

删除

从文本域中删除选中的文字。

选择全部

选中页面中的所有文字及其他元素。

在当前页查找...

在浏览器窗口的底部显示查找工具条。在工具条中输入您想查找的文字， 随着您的输入，&brandShortName; 会自动找到并选中当前页面中出现的第一个匹配文字。单击 查找下一个或者查找上一个来找到页面内其他位置的匹配文字，单击高亮显示可以让页面内匹配的文字以高亮显示出来。

再次查找

查找在使用在当前页查找...命令时指定的文字的下一个匹配项。

&pref.pluralCaps;

显示 &pref.pluralCaps; 窗口, 在这里您可以改变 &brandShortName; 的许多&pref.plural;。

查看

工具栏

导航工具栏

切换导航工具栏的显示与否。导航工具栏通常包含标准的导航按钮（后退、前进、停止等）。

书签工具栏

切换书签工具栏的显示与否。书签工具栏包括指向不同网站的书签。

定制...

显示工具栏定制对话框。在这里您可以在工具栏上拖放按钮。您也可以通过右击按下 &ctrlKey; 键， 单击导航工具栏并选择定制...来显示此对话框。

状态栏

切换状态栏显示与否。状态栏显示在窗口的底部， 包含一些同页面相关的有用信息，但可能不是每个人都需要它。

侧栏

侧栏可以显示您的书签或浏览历史。您可以通过单击右上角的 X 来关闭侧栏。

书签

显示书签侧栏，这可以让您随时都能看到书签。如果您经常使用书签， 这会非常有用。

历史

显示历史侧栏，它能够显示您已经访问过的那些网址。如果想改变记忆网址的天数， 选择 &pref.menuPath; 并选择隐私面板。

停止

停止下载您正在浏览的当前页面。其效果等同于单击工具栏上的停止按钮。

重新载入

获取当前页面的最新版本。这等同于单击工具栏上的重新载入按钮。

文本大小

如果网页上的文字太大或太小，您可以把它改成更舒服的字体大小。

放大文本

以大一号尺寸来显示网页上的文字。同样的操作您也可以用鼠标来完成（参见鼠标快捷键以获取更多信息）。

缩小文本

以小一号的尺寸来显示网页文字。同样的操作也可以用鼠标来完成（参见鼠标快捷键以获取更多信息）。

正常

用正常大小显示网页文字。

页面风格

这里允许您为当前页面选一个风格。默认情况下会选定“基本页面风格”， 除非作者指定了另外的默认风格。

无风格

取消页面的格式。

基本页面风格

用作者指定的普通风格来显示页面。

如果作者指定了额外的页面风格，&brandShortName; 会把它们列在此处以供选择。

字符编码

这里可以手动改变网页的字符编码。通常，&brandShortName; 会自动选择字符编码。

页面源代码

显示当前页面的源代码。

全屏

用全屏模式显示页面。如果在此模式下导航栏也被隐藏了， 您还可以用这种方法返回窗口模式，按下 F11。

历史

“历史”菜单里是一个最新访问过的网址和打开的标签页的列表。

后退

在历史列表中后退一个页面。这等同于单击工具栏上的后退按钮。

前进

在历史列表中前进一个页面。这等同于单击工具栏上的前进按钮。

主页

显示您的主页。这等同于单击导航栏上的主页按钮。要想改变主页， 选择 &pref.menuPath;，打开“主要”面板； 您可以指定一个空白页、当前页（或一组标签页）、 一个书签、或是一个手工输入的地址来作为主页。

最近关闭的标签页

在这里通过从列表中选择相应的项目，您可以恢复打开当前窗口中最新关闭的标签页。参阅主题关闭和恢复标签页获得更多信息。

显示于侧栏

显示历史侧栏，它会跟踪你访问过的网站。要改变记忆网站的天数，选择 &pref.menuPath; 并选择隐私面板。

书签

这一菜单包括所有您存储的书签。

将本页加入书签

把当前页面加到书签中。选择这一命令会弹出一个对话框， 让您为这个书签设定标题并指定把它存储在什么地方。

订阅此页...

显示该网站提供的新闻源的预览。通过预览页，您可以使用实时书签、您的电脑上的新闻源阅读程序或者一个网络服务订阅这个新闻源。如果您在 &pref.menuPath; 的收取点面板中选择了总是使用实时书签或者某个新闻源阅读器，则预览页会被跳过。

将全部标签页加为书签...

将当前窗口的所有标签页加到一个新的书签文件夹中。会显示一个对话框让您选择文件夹的名称并指定它要存储到什么位置。

管理书签...

显示书签管理器窗口，在这里您可以修改书签， 可以对书签进行排序、重命名以及修改它的属性， 还可以添加、删除或者移动书签的位置。

工具

网页搜索

高亮显示网页搜索栏。您可以在其中输入您想搜索的关键词。

下载项

打开下载管理器，在这里可以看到正在进行的或是已经完成的下载项目。

附加组件

打开附加组件管理器，在这里您可以查看、安装、设置、更新和卸载扩展及主题。要获得更多信息，参阅帮助主题附加组件。

错误控制台

打开错误控制台，它可以跟踪 JavaScript 代码中的错误。JavaScript 是一种脚本语言，通常用于构建网页。程序员用 JavaScript 来让网页更具交互性；它经常被用于表单验证和选择按钮。

页面信息

显示当前页面的更多信息，比如文档类型、编码、文件大小以及安全信息。 这个窗口还会显示该页面中所用的媒体和链接列表。

清除私有数据...

清除在“清除私有数据”对话框中被选中的项，该对话框在 &pref.menuPath; 的隐私面板中。 默认会显示一个确认对话框。

&pref.pluralCaps;

显示 &pref.pluralCaps;窗口，在这里您可以改变 &brandShortName; 的许多&pref.plural;。

窗口

最小化

最小化当前窗口。

缩放

放大或缩小当前窗口。

（打开窗口）

显示所有打开窗口的标题。

帮助

帮助内容 &brandShortName; 帮助

打开 &brandShortName; 帮助窗口，里面的内容对您浏览网页非常有帮助。

致 Internet Explorer 用户

这个帮助窗口中的信息可以帮助 Internet Explorer 用户转换到 &brandShortName;。

发行说明

提供安装、卸载和设置 &brandShortName; 的相关信息，还包括其他重要信息以及后发布的通告。需要可用的 Internet 连接。

汇报伪造网站...

在这里汇报您确信为伪造或者“钓鱼”的网站。&brandShortName; 的反钓鱼保护功能由 Google 提供，您汇报的信息会发送给 Google 用于改进这个功能。您的汇报内容会受到 Google 的隐私策略的保护。需要可用的 Internet 连接。

汇报不兼容网站...

在这里将 &brandShortName; 中不能正常工组的网站汇报给 &brandShortName; 的开发者。

检查更新...

显示一个对话框检查 &brandShortName; 的更新，当有任何可用的更新时询问您是否要下载。下载更新后，会有对话框提示您重新启动 &brandShortName; 以便安装更新。注意，当正在下载一个更新或者准备好安装一个更新时，这个菜单项目的名称会发生变化。

关于 &brandFullName;

显示一个对话框，其中是关于 &brandShortName; 的信息，包括 当前的版本以及一个简短的致谢名单。

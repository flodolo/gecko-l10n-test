&brandShortName;

Acerca de &brandFullName;

Muestra un cuadro de diálogo con información sobre &brandShortName;, incluyendo la versión actual y una breve lista de créditos.

Preferencias…

Muestra la ventana &pref.pluralCaps;, donde puede cambiar muchas &pref.plural; en &brandShortName;.

Salir de &brandShortName;

Cierra todas las ventanas de &brandShortName;, cancela todas las descargas activas y sale por completo de &brandShortName;. Si selecciona esta opción de menú con más de una pestaña o ventana abiertas, &brandShortName; le preguntará para confirmar su elección y evitar salidas accidentales. Puede desactivar esta advertencia desmarcando la casilla Advertirme al cerrar múltiples pestañas en el panel Pestañas de &pref.menuPath;. &brandShortName; también le advertirá si está descargando en ese momento algún archivo.

Archivo

Nueva ventana

Abre una nueva ventana de &brandShortName;.

Nueva pestaña

Abre una nueva pestaña del navegador en la ventana actual de &brandShortName;.

Abrir dirección…

Sitúa el cursor en la barra de URLs de &brandShortName; para que pueda escribir una nueva dirección a la que dirigirse.

Abrir archivo…

Muestra un diálogo para abrir archivos donde puede seleccionar un archivo de sus discos o redes locales. Puede ver muchos tipos de archivos en &brandShortName;, incluyendo archivos HTML/XML, archivos de imágenes, archivos de texto y otros.

Cerrar (ventana)

Cierra la ventana actual.

Cerrar pestaña

Cierra la pestaña actual y selecciona la siguiente pestaña a la derecha. Este elemento de menú sólo está visible si hay más de una pestaña abierta.

Guardar página como…

Guarda la página que está visualizando actualmente. Tiene la opción de guardar la página completa (con gráficos), sólo la página HTML o únicamente el texto de la página.

Enviar enlace…

Abre una ventana de redacción de correo electrónico usando el programa de correo electrónico predeterminado de su ordenador, de modo que pueda enviar un mensaje con un enlace a la página actual.

Configurar página…

Muestra un diálogo de configuración de página donde pueden especificarse opciones de impresión tales como márgenes, cabeceras y pies, y orientación de página.

Vista preliminar (sólo Windows y Linux)

Muestra una vista preliminar de cómo aparecerá el documento actual al imprimirlo. Refleja las opciones elegidas en el cuadro de configuración de página y los propósitos del autor para la página.

Imprimir…

Muestra el diálogo Imprimir, donde puede especificar el número de copias impresas, etc. Pulse Aceptar para imprimir la página.

Importar…

Abre el asistente de importación, que le permite importar &pref.plural;, marcadores, historial, contraseñas y otros datos de navegadores como Microsoft Internet Explorer, Netscape, Mozilla u Opera.

Trabajar sin conexión

Alterna entre modo con y sin conexión. El modo sin conexión le permite ver páginas web que ha visitado previamente sin estar conectado a Internet.

Salir Salir

Cierra todas las ventanas de &brandShortName;, cancela todas las descargas activas y sale por completo de &brandShortName;. Si selecciona esta opción de menú con más de una pestaña o ventana abiertas, &brandShortName; le preguntará para confirmar su elección y evitar salidas accidentales. Puede desactivar esta advertencia desmarcando la casilla Advertirme al cerrar múltiples pestañas en el panel Pestañas de &pref.menuPath;. &brandShortName; también le advertirá si está descargando en ese momento algún archivo.

Editar

Deshacer

Deshace la última acción en un campo de texto. Si más tarde decide que no quiere deshacer una acción, utilice el comando Rehacer.

Rehacer

Repite la última orden Deshacer.

Cortar

Corta el texto seleccionado de un campo de texto y lo guarda en el portapapeles. Pegue el texto en cualquier otra parte usando la opción Pegar.

Copiar

Copia el texto seleccionado en el portapapeles. Pegue el texto en un campo de texto usando la opción Pegar.

Pegar

Pega el texto guardado en el portapapeles en un campo de texto.

Borrar

Borra el texto seleccionado de un campo de texto.

Seleccionar todo

Selecciona todo el texto y otros elementos de la página.

Buscar en esta página…

Muestra la barra de herramientas de búsqueda al pie de la ventana del navegador. Introduzca el texto que desea buscar en la barra de herramientas, y a medida que escriba &brandShortName; encontrará y seleccionará automáticamente la primera ocurrencia del texto introducido en la página. Pulse Buscar siguiente o Buscar anterior para encontrar el texto en otras posiciones de la página, y pulse Resaltar para destacar todas las ocurrencias del texto en la página.

Repetir la búsqueda

Busca la siguiente ocurrencia del texto especificado al usar la opción Buscar en esta página….

&pref.pluralCaps;

Muestra la ventana de &pref.pluralCaps;, donde puede cambiar muchas &pref.plural; de &brandShortName;.

Ver

Barras de herramientas

Barra de herramientas de navegación

Alterna la visibilidad de la barra de herramientas de navegación, que normalmente contiene los botones estándar de navegación (Anterior, Siguiente, Parar, etc.).

Barra de herramientas de marcadores

Alterna la visibilidad de la barra de herramientas de marcadores, que contiene marcadores a diversos sitios.

Personalizar…

Muestra el diálogo de personalización de la barra de herramientas. Puede arrastrar y soltar botones entre la barra de herramientas y aquí. También puede pulsar con el botón derechopulsar &ctrlKey; y hacer clic en la barra de navegación y seleccionar Personalizar… para mostrar este diálogo.

Barra de estado

Alterna la visibilidad de la barra de estado, ubicada en la parte inferior de la ventana. La barra de estado muestra información útil sobre la página pero puede no ser necesaria para todo el mundo.

Panel lateral

El panel lateral puede mostrar sus marcadores o el historial de navegación. Siempre puede cerrar el panel lateral pulsando en la X en la esquina superior derecha.

Marcadores

Muestra el panel lateral de marcadores, que mantiene visibles sus marcadores todo el tiempo. Esto puede resultar muy útil si usa los marcadores a menudo.

Historial

Muestra el panel lateral del historial, que mantiene registrados los sitios que ha visitado. Para cambiar cuántos días debe recordarse un sitio, seleccione &pref.menuPath; y seleccione el panel Privacidad.

Parar

Detiene la carga de la página que está visitando actualmente. Es equivalente a pulsar el botón Parar de la barra de herramientas.

Recargar

Obtiene la versión más actualizada de la página actual. Esto es equivalente a pulsar el botón Recargar en la barra de herramientas.

Tamaño del texto

Si el texto en una página web es demasiado pequeño o demasiado grande, puede cambiarlo a un tamaño más cómodo.

Incrementar

Muestra el texto una talla mayor en la página web. También puede hacer esto con el ratón (vea Atajos del ratón para más detalles).

Disminuir

Hace más pequeño el texto de la página web. Puede hacer esto mismo también con el ratón (vea Atajos del ratón para más detalles).

Normal

Muestra el texto en la página web con su tamaño normal.

Estilo de página

Le permite seleccionar un estilo para la página web en curso. Por omisión, se selecciona "Estilo de página básico" a menos que el autor especifique un valor predeterminado diferente.

Sin estilo

Elimina el formato de estilo de la página.

Estilo de página básico

Muestra la página con un estilo general especificado por el autor.

Si el autor ha especificado estilos de página adicionales, &brandShortName; los listará aquí para que los pueda seleccionar.

Codificación de caracteres

Le permite cambiar manualmente la codificación de caracteres en una página web. Normalmente, &brandShortName; hace esto automáticamente.

Código fuente de la página

Muestra el código fuente del documento actual.

Pantalla completa

Muestra la ventana en pantalla completa. Si tiene la barra de herramientas de navegación oculta, puede restaurar la ventana en cualquier caso pulsando F11.

Historial

El menú Historial contiene una lista de los últimos sitios web visitados y pestañas cerradas.

Anterior

Navega una página atrás en el historial. Esto es lo mismo que pulsar el botón Anterior en la barra de herramientas.

Siguiente

Navega una página adelante en el historial. Esto es lo mismo que pulsar el botón Siguiente en la barra de herramientas.

Inicio

Muestra su página de inicio. Esto es lo mismo que pulsar el botón Inicio en la barra de herramientas de navegación. Para cambiar la página de inicio, seleccione &pref.menuPath; y abra el panel Principal. Puede indicar que se use una página en blanco, la página actual (o grupo de pestañas), un marcador, o una dirección introducida manualmente.

Pestañas cerradas recientemente

Le permite restaurar una pestaña que haya cerrado recientemente en la ventana actual seleccionándola de la lista. Vea el tema Cerrar y restaurar pestañas para más información.

Mostrar en panel lateral

Muestra el panel lateral Historial, que registra los sitios que ha visitado. Para cambiar cuántos días debe recordarse un sitio, seleccione &pref.menuPath; y escoja el panel Privacidad panel.

Marcadores

Este menú contiene todos sus marcadores guardados.

Añadir esta página a marcadores…

Añade la página actual a sus marcadores. Aparecerá un diálogo que le permitirá elegir un título para el marcador y especificar dónde quiere que se guarde.

Suscribirse a esta página…

Muestra una vista preliminar del canal ofrecido por el sitio en curso. Desde la vista preliminar, puede suscribirse al canal usando un marcador dinámico, un agregador de canales en su ordenador, o un servicio web. Si ha elegido suscribirse siempre usando un marcador dinámico o un agregador de canales en el panel Canales de &pref.menuPath;, se omitirá la vista preliminar.

Añadir todas las pestañas a marcadores

Añade todas las pestañas en la ventana actual a una nueva carpeta de marcadores. Se mostrará un diálogo que le permite escoger el nombre de la carpeta y especificar dónde quiere guardarla.

Organizar marcadores…

Muestra el administrador de marcadores, una ventana donde puede modificar sus marcadores. Puede ordenar, renombrar y cambiar las propiedades de sus marcadores así como añadirlos, eliminarlos y moverlos.

Herramientas

Buscar en la web

Sitúa el cursor en la caja de búsqueda en la web. Puede escribir ahí los términos que desee buscar en la web.

Descargas

Abre el administrador de descargas, donde puede ver las descargas actuales así como las completadas.

Complementos

Abre el administrador de complementos, donde puede ver, instalar, configurar, actualizar y desinstalar sus extensiones y temas. Para más información, vea el tema de la ayuda Complementos.

Consola de error

Abre la consola de error, que registra los problemas con el código JavaScript. JavaScript es un lenguaje de scripting usado habitualmente para construir páginas web. Los programadores usan JavaScript para hacer las páginas web más interactivas; JavaScript se usa a menudo para validar dinámicamente formularios y seleccionar botones.

Información de página

Muestra amplia información sobre la página web actual, tal como el tipo de documento, codificación, tamaño e información de seguridad. El diálogo también muestra listas de medios y enlaces usados en la página.

Limpiar información privada…

Limpia los elementos seleccionados en el diálogo Limpiar información privada del panel Privacidad de &pref.menuPath;. Por omisión, muestra un cuadro de diálogo de confirmación.

Informar de falsificación de web…

Le permite avisar de sitios que crea que son fraudulentos o de "phising". La protección contra phising en &brandShortName; se apoya en Google, y la información que envía será utilizada por Google para mejorar esta característica. Su informe será anónimo, de acuerdo con la política de privacidad de Google. Esta característica requiere una conexión a Internet activa.

Informar de sitio web incompatible

Buscar actualizaciones…

Muestra un diálogo que busca actualizaciones para &brandShortName; y pregunta si se quieren descargar en caso de que haya alguna disponible. Después de que se haya descargado una actualización, el diálogo le pide reiniciar &brandShortName; para que se instale la actualización. Observe que el nombre de esta opción de menú cambiará cuando se está descargando una actualización o está lista para ser instalada.

Denna ordlista ger endast en kort beskrivning av ord och termer och är inte tänkt att ge en fullständig eller fackmannamässig beskrivning av termerna, inte heller de säkerhets- eller personvärnsmässiga följderna av de tekniker som beskrivs.

Ordlista

Adressfältet
Fältet (och tillhörande knappar) nära den övre delen av ett &brandShortName;-fönster där du kan skriva in en URL eller sökord.
Aktivt bokmärke
Ett särskilt sorts bokmärke som fungerar som en mapp med länkar för en kanal. Du kan skapa ett aktivt bokmärke genom att besöka en webbplats som har en kanal, klicka på ikonen för aktivt bokmärke i Adressfältet och välja en plats för det.
autentisering
Användandet av lösenord, certifikat, PIN-koder eller annan information för att styrka en identitet över ett datanätverk.
avlyssning
Hemlig avlyssning av information skickad över ett nätverk, som görs av en enhet som inte är avsedd att få del av informationen.
bokmärke
En sparad webbsidesadress (URL) som du enkelt kan besöka genom att klicka på bokmärkets ikon på det personliga verktygsfältet eller genom att välja bokmärkets namn i Bokmärken-menyn.
Bokmärkesfältet
Ett anpassningsbart verktygsfält som vanligtvis visas just under Adressfältet i &brandShortName;. Det innehåller standardknappar som du kan lägga till och ta bort, såsom knappar för dina favoritbokmärken, eller mappar som innehåller grupper av bokmärken.
cache
En samling av webbsidor som har sparats på datorns hårddisk eller i dess RAM-minne. Webbläsaren samlar på sig dessa kopior när du surfar på webben. När du klickar på en länk eller skriver in en URL för att hämta en särskild webbsida som redan finns i cachen jämförs den sparade kopian med originalet. Om inga ändringar har skett kommer webbläsaren att använda den cachade kopian i stället för att hämta sidan på nytt, för att på så sätt spara tid.
certifikat
Den digitala motsvarigheten till ID-kort. Ett certifikat anger namnet på en person, ett företag eller annan enhet och intygar att den öppna nyckel som följer med certifikatet tillhör denne. När du signerar ett meddelande eller annan data kommer den digitala signaturen att skapas med hjälp av den privata nyckel som svarar mot den öppna nyckeln i ditt certifikat.
dekryptering
Processen att återskapa data som har krypterats. Se även kryptering.
digitalt ID
Se certifikat.
Extensible Markup Language (XML)
En öppen standard för beskrivning av data. Till skillnad från HTML tillåter XML utvecklaren av en webbsida att definiera speciella taggar. För mer information, se det webbaserade W3C-dokumentet Extensible Markup Language (XML).
File Transfer Protocol (FTP)
En standard som låter användare överföra filer från en dator till en annan över ett nätverk. Du kan använda din webbläsare för att hämta filer via FTP.
FIPS PUBS 140-1
Federal Information Processing Standards Publications (FIPS PUBS) 140-1 är en amerikansk standard för implementeringar av kryptografiska moduler--dvs. hårdvara eller programvara som krypterar och dekrypterar data eller utför andra kryptografiska handlingar (såsom skapandet eller verifierandet av digitala signaturer). Många produkter som säljs till den amerikanska staten måste följa en eller flera av FIPS-standarderna.
hemsida
Sidan som din webbläsare visar vid starten eller när du klickar på knappen Startsida. Används också som namn på huvudsidan för en webbplats, från vilken du kan utforska övriga delar av webbplatsen.
Hypertext Markup Language (HTML)
Dokumentformatet som används för webbsidor. HTML-standarden definierar taggar eller koder som används för att definiera layouten för text, teckensnitt, stilar, bilder och andra element som utgör en webbsida.
huvudlösenord
Ett lösenord som används för att skydda sparade lösenord och annan personlig data. &brandShortName; kommer att fråga dig om huvudlösenordet när du vill använda denna data. Om du har flera säkerhetsenheter kommer varje säkerhetsenhet att behöva ett eget huvudlösenord.
insticksmodul
Insticksmoduler ger utökade möjligheter för din webbläsare att spela upp ljud eller filmklipp. I motsats till andra typer av hjälpprogram så installeras insticksmoduler i mappen Plugins i webbläsarens installationsmapp, och kan oftast öppnas inne i själva webbläsaren (internt). Med en ljud-insticksmodul kan du t.ex. lyssna på ljudfiler på webbsidor eller i e-postmeddelanden. Macromedia Flash Player och Java är båda exempel på insticksmoduler.
Internet
Ett världsomspännande nätverk av miljontals datorer som kommunicerar med varandra med hjälp av standardprotokoll som TCP/IP. Från början utvecklat för USA:s försvar 1969, utökades Internet till att innefatta utbildnings- och forskningsinstitutioner, och i slutet av 90-talet miljoner företag, organisationer och privatpersoner. Idag används Internet för e-post, surfning på webben, snabbmeddelanden, diskussionsgrupper och många andra ändamål.
Internet protocol address (IP-adress)
Adressen för en dator på ett TCP/IP-nätverk. Varje dator på Internet har en IP-adress. Klienter har antingen en fast IP-adress eller en de har fått varje gång de anslutits till nätverket (så kallad dynamisk IP-adress). IP-adresser skrivs som siffror i fyra grupper, så här: 204.171.64.2.
Java
Ett programmeringsspråk utvecklat av Sun Microsystems. Ett enkelt Java-program kan köras på många olika sorters datorer, på så sätt slipper man utveckla olika versioner av program för varje datorplattform. Din webbläsare kan automatiskt hämta och köra Java-program (även kallad appletar) om du har en instickmodul för Java installerad.
JavaScript
Ett skriptspråk vanligen använt vid skapandet av webbsidor. Programmerare använder JavaScript för att göra webbsidor mer interaktiva, t.ex. för att visa knappar och formulär. JavaScript kan användas med Java, men de är tekniskt sett helt separerade. Java krävs inte för att JavaScript ska fungera.
kaka
En liten bit information som lagras på din dator av en del webbplatser. När du besöker en sådan webbplats kommer den att begära av din webbläsare att få lagra en eller flera kakor på din hårddisk. När du senare återvänder till platsen kommer din webbläsare att skicka kakan eller kakorna som hör till webbplatsen. Kakor hjälper webbplatser att hålla koll på information om dig, såsom innehållet i en varukorg. Du kan i inställningarna för kakor ange hur du vill att kakor används.
kanal
En XML-webbsida som innehållet en lista på länkar till andra webbsidor. Speciella program kan läsa av dessa kanaler och skapa en lista med rubriker från länkarna, och automatiskt uppdatera listan när den ändras. Nyhetswebbplatser använder kanaler för att snabbt publicera de senaste nyheterna, och personliga webbaserade journaler använder ofta kanaler för att snabbt underrätta besökare om nya inlägg. Se även aktivt bokmärke.
klient
Programvara (t.ex. en webbläsare) som skickar förfrågningar och mottar information från en server som vanligtvis körs på en annan dator. En dator som används till att köra klient-programvara på kan också benämnas klient.
kryptering
Processen att förvränga information så att dess innebörd skyddas. Exempelvis kan en krypterad förbindelse mellan datorer göra det väldigt svårt för obehöriga att återskapa eller dekryptera information som strömmar över förbindelsen. Krypterad information kan endast dekrypteras av någon som har tillgång till rätt nyckel.
kryptografi
Tekniken och praktiserandet av kryptering och dekryptering av information. Exempelvis används kryptografi till att förvränga och återskapa information som strömmar mellan kommersiella webbplatser och din webbläsare.
Lösenordshanteraren
Den del av webbläsaren som hjälper dig att komma ihåg dina användarnamn eller lösenord, och som skriver i dem för dig automatiskt när du återbesöker sådana platser.
Navigeringsfältet
Verktygsfältet i den övre delen av webbläsarfönstret som innehåller Bakåt och Framåt-knapparna.
PKCS #11
Den standard baserad på öppen nyckel-kryptografi som styr säkerhetsenheter, såsom smartkort.
privat nyckel
Den ena av ett par av nycklar som används för öppen nyckel-kryptografi. Den privata nyckeln lagras undangömt och används för att dekryptera data som har krypterats med den tillhörande öppna nyckeln.
proxy
Ett mellanliggande eller medlande program som agerar både som en server och som en klient för att göra förfrågningar på uppdrag av andra klienter.
Secure Sockets Layer (SSL)
Ett protokoll som tillåter ömsesidig autentisering mellan en klient och en server med syftet att etablera en bekräftad och krypterad förbindelse. SSL körs ovanpå TCP/IP och under HTTP, LDAP, IMAP, NNTP och andra högnivåprotokoll. Den nya Internet Engineering Task Force(IETF)-standarden kallad Transport Layer Security (TLS) är baserad på SSL. Se även autentisering och kryptering.
server
Programvara (t.ex. mjukvara som tillhandahåller webbsidor) som mottar efterfrågningar från, och skickar information till en klient, som vanligtvis körs på en annan dator. En dator på vilken serverprogramvara körs kallas också för server.
Statusfält
Det verktygsfält som visas i nedre delen av ett &brandShortName;-fönster. Det visar ikoner för statusen på sidan och adressen för fokuserade länkar
säker webbplats
En webbplats som använder kryptering vid anslutningar med &brandShortName; för att förhindra obehöriga Internetanvändare från att se överförd data. När du besöker säkra webbplatser kommer &brandShortName; att visa en låsikon i Statusfältet och Adressfältet. &brandShortName; visar också webbplatsens domännamn i Statusfältet (för att förhindra illasinnade webbplatser från att stjäla data) och ändrar bakgrundsfärgen i Adressfältet till gul.
En del webbplatser innehåller både säker och osäker data; den osäkra datan är vanligen inte av känslig art, även om det ibland kan förekomma. För sådana platser kommer &brandShortName; att visa ett streck igenom låsikonen i Statusfältet och Adressfältet och kommer inte att visa webbplatsens domännamn i Statusfältet. På så sätt vet du att den aktuella webbsidan inte är fullständigt säker.
sökmotor
Ett webbaserat program som låter användare söka efter och hämta specifik information från webben. Sökmotorn kan söka bland all text i webbdokument, i en lista med nyckelord, eller använda bibliotekarier som granskar webbdokument och som manuellt upprättar ett index över sidorna. En användare skriver vanligtvis in ett ord eller en fras i en sökruta och sökmotorn visar därefter länkar till relevanta webbsidor.
TCP/IP (Transmission Control Protocol/Internet Protocol)
Ett Unix-protokoll som används för att ansluta datorer som kan köra olika operativsystem. TCP/IP är ett viktigt Internetprotokoll och har blivit en global standard.
TLS
Se Secure Sockets Layer (SSL).
Uniform Resource Locator (URL)
Den standardiserade adress som talar om för webbläsaren hur den ska lokalisera en fil eller annan resurs på webben, t.ex.: http://www.mozilla.org. Du kan skriva in URL:er i webbläsarens Adressfält för att gå till webbsidor. URL:er används också i länkar på webbsidor som du kan klicka på för att gå till andra sidor. Även känd som Internetadress eller webbadress.
webbplats
En grupp av relaterade webbsidor som är ihoplänkade av hyperlänkar och hanteras av ett företag, organisation eller individ. En webbplats kan innehålla text, grafik, ljud och video-filer och länkar till andra webbplatser.
webbsida
Ett dokument på webben som specificerats av en unik adress eller URL och som kan innehålla text, hyperlänkar och grafik.
World Wide Web
Även känd som webben. En del av Internet som består av webbsidor lagrade på webbservrar och som visas av klienter kallade webbläsare (såsom &brandShortName;).

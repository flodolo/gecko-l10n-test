Acest glosar este furnizat doar pentru informarea dumneavoastră; nu este menit să fie privit ca o descriere completă sau autoritativă a termenilor definiți mai jos sau a ramificațiilor de confidențialitate și/sau securitate ale tehnologiilor descrise.

Glosar

autentificare
A folosi o parolă, un certificat, un număr personal de identificare (PIN) sau a altei informații pentru a valida o identitate într-o rețea de calculatoare.
semn de carte
O adresă (URL) stocată la care puteți merge cu ușurință clicând iconița sa în bara pentru semne de carte sau alegând numele semnului de carte din meniul Semne de carte.
Bara pentru semne de carte
Este bara de unelte personalizabilă care apare chiar sub bara adresei la prima folosire a &brandShortName;. Conține butoane pentru semnele dumneavoastră de carte (sau dosare care grupează semne de carte) pe care le puteți adăuga sau șterge.
cache
O colecție de copii ale unor pagini web stocate pe discul calculatorului dumneavoastră sau în memoria cu acces aleator (RAM). &brandShortName; acumulează aceste copii pe măsură ce dumneavoastră răsfoiți paginile de pe internet. Când clicați o legătură sau tastați o adresă URL pentru a aduce o pagină anume pentru care cache-ul conține deja o copie, &brandShortName; compară copia stocată cu originalul. Dacă nu au intervenit schimbări, &brandShortName; folosește copia stocată, evitând readucerea originalului, economisind astfel timp pentru dumneavoastră.
certificat
Echivalentul digital al unei cărți de identitate. Un certificat specifică numele unui individ, al unei companii sau al unei entități și certifică faptul că o cheie publică, care este inclusă în certificat, aparține acelei entități. Când semnați digital un mesaj sau alte date, semnătura digitală pentru acel mesaj este creată cu ajutorul unei chei private care corespunde cheii publice din certificatul dumneavoastră.
client
Un program (cum ar fi un navigator de internet) care trimite cereri către un server și primește informații de la acesta. Serverul este de obicei situat pe un alt calculator decât al dumneavoastră. Un calculator pe care rulează un program client este de asemenea definit ca un client.
cookie
O mică cantitate de informație stocată pe calculatorul dumneavoastră de unele saituri web. Când vizitați un asemenea sait, acesta cere &brandShortName; să plaseze una sau mai multe cookie-uri pe hard-discul dumneavoastră. Ulterior, când reveniți pe sait, &brandShortName; trimite saitului cookie-urile care îi aparțin. Cookie-urile ajută saiturile să păstreze anumite informații legate de dumneavoastră, cum ar fi conținutul coșului de cumpărături. Puteți ajusta opțiunile dumneavoastră pentru aceste cookie-uri pentru a controla cum sunt folosite și câtă informație sunteți dispus(ă) să permiteți saiturilor să stocheze în ele.
criptografie
Arta și practica de a coda (cripta) și decoda (decripta) informația De exemplu, tehnici de criptografie sunt utilizate pentru a coda și decoda informațiile care sunt transmise între saituri comerciale web și &brandShortName;.
decriptare
Procesul de a decoda datele care au fost criptate. Citiți și despre criptare.
ID digital
Vedeți certificat.
criptare
Procesul de codare a informație într-un fel care îi maschează conținutul. De exemplu, conexiunile criptate între calculatoare fac foarte dificilă decodarea sau decriptarea de către terți a informației care circulă pe o conexiune. Informația criptată poate fi decriptată doar de cineva care deține cheia potrivită.
Limbaj de marcare extensibil (XML)
Un standard deschis pentru a descrie datele. Spre deosebire de HTML, XML permite dezvoltatorilor unei pagini web să definească etichete speciale. Pentru mai multe informații, puteți citi documentul W3C disponibil online, în engleză Extensible Markup Language (XML).
feed
O pagină XML care conține o listă de legături către alte pagini. Programe speciale pot citi aceste fluxuri (în engleză „feeds”) pentru a crea o listă de capete de afiș din legături, actualizând automat lista pe măsură ce aceasta se schimbă. Saiturile de știri folosesc fluxuri pentru a publica rapid ultimele capete de afiș. Jurnalele personale online folosesc adesea fluxuri pentru a notifica vizitatorii când apar noi intrări. Citiți și despre semn de carte actualizabil.
Protocol de transfer a fișierelor(FTP)
Un standard care permite utilizatorilor să transfere fișiere de pe un calculator pe altul din rețea. Puteți folosi &brandShortName; pentru a transfera fișere folosind FTP.
FIPS PUBS 140-1
Federal Information Processing Standards Publications (FIPS PUBS) 140-1 este un standard al guvernului SUA pentru implementarea modulelor de criptografie — hardware sau software care criptează sau decriptează date și execută alte operațiuni criptografice (cum ar fi crearea sau verificarea semnăturilor digitale). Multe produse vândute guvernului SUA trebuie să respecte unul sau mai multe standarde FIPS.
pagina de start
Este pagina pe care &brandShortName; o afișează de fiecare dată când e lansat sau când dumneavoastră clicați butonul Acasă. Folosit de asemenea pentru a referi pagina principală a unui sait web de la care puteți explora restul saitului.
Limbajul de marcare hipertext (HTML)
Formatul de document utilizat pentru pagini web. Standardul HTML definește etichete sau coduri folosite pentru a defini amplasamentul textului, fonturile, stilul, imaginile și alte elemente care compun o pagină web.
Internet
O rețea mondială de milioane de calculatoare care comunică între ele folosind protocoale standard cum ar fi TCP/IP. Inițial dezvoltat pentru armata SUA în 1969, internetul a crescut prin includerea instituțiilor de educație și cercetare și la sfârșitul anilor '90, milioane de afaceri, organizații și individizi. Astăzi Internetul este folosit pentru corespondență electronică, navigarea în World Wide Web (Rețeaua mondială), mesagerie instantă, liste de corespondență și grupuri de discuții precum și pentru alte scopuri.
Adresa pentru protocolul internet (adresă IP)
Adresa unui calculator într-o rețea TCP/IP. Fiecare calculator conectat la internet are o adresă IP. Clienții au fie o adresă IP permanentă sau una flotantă desemnată de fiecare dată când se conectează la rețea.
Java
Un limbaj de programare dezvoltat de Sun Microsystems. Un singur program Java poate rula pe mai multe tipuri de calculatoare, evitând astfel nevoia de a crea versiuni separate ale programului pentru fiecare tip de calculator. Dupa ce Java este instalat, &brandShortName; poate descărca și rula programe Java automat (programele mai sunt numite și applet-uri).
JavaScript
Un limbaj scriptic folosit adesea în construcția paginilor web. Programatorii folosesc JavaScript pentru a face paginile web mai interactive; JavaScript este adesea utilizat pentru a valida dinamic formularele și a selecta butoane. JavaScript poate fi utilizat cu Java, dar acesta este din punct de vedere tehnic un limbaj separat. Java nu este necesar pentru ca JavaScript să funcționeze corect.
Semn de carte actualizabil
Un tip special de semn de carte care se comportă ca un dosar. El conține legături dintr-un flux de informații. Puteți crea un semn de carte actualizabil vizitând un sait cu un flux, clicând pe iconița portocalie a semnului actualizabil din bara adresei, în partea dreaptă, și selectând ce flux doriți să folosiți.
Bara adresei
Câmpul (și butoanele asociate) din partea de sus a ferestrei &brandShortName; unde puteți tasta o adresă URL sau termeni de căutare.
parolă generală
O parolă care este folosită pentru a proteja parolele salvate și alte date private. &brandShortName; vă va cere această parolă pentru confirmare când doriți să accesați date private. Dacă aveți mai multe dispozitive de securitate, fiecare dispozitiv de securitate va necesita o parolă generală separată.
Bară de navigare
Bara din partea de sus a ferestrei &brandShortName; care include butoanele Înapoi și Înainte.
Gestionar de parole
Partea din &brandShortName; care vă ajută să salvați unele sau toate datele de autentificare, cum sunt numele de utilizator și parola stocându-le pe calculatorul dumneavoastră după ce le-ați introdus o dată și introducându-le automat în formulare când vizitați saiturile din nou.
PKCS #11
Standardul de criptografie prin cheie publică care guvernează dispozitive de securitate cum ar fi smart card-uri.
Plugin
Numele provine din limba engleză, însemnând o părticică de aplicație care nu poate funcționa singură, dar înglobată într-o aplicație de sine stătătoare îi adaugă funcționalități. Plugin-urile adaugă noi abilități în &brandShortName;, cum ar fi abilitatea de a reda clipuri audio sau video. Spre deosebire de alte aplicații ajutătoare, un plugin se instalează în directorul de plugin-uri din directorul principal de instalare al &brandShortName; și poate fi deschis de &brandShortName; (intern). De exemplu, un plugin audio vă permite să ascultați fișiere audio de pe o pagină web sau dintr-un mesaj email. Macromedia Flash Player și Java sunt ambele exemple de aplicații de tip plugin.
cheie privată
Una din perechea de chei folosite în criptografia bazată pe cheie publică. Cheia privată este păstrată secretă și este folosită pentru decriptarea datelor care au fost criptate cu cheia publică asociată.
proxy
Un program intermediar care se comportă și ca server și ca client pentru a face cereri în numele altor clienți.
motor de căutare
Un program cu o interfață disponibilă pe web care permite utilizatorilor să caute informații în World Wide Web. Motorul de căutare poate căuta în tot textul documentelor web sau într-o listă de cuvinte cheie; de asemenea, el poate să folosească librării care revizuiesc documentele web și le indexează apoi manual pentru cereri ulterioare. În mod normal, utilizatorul tastează un cuvânt sau o frază într-o căsuță de căutare și motorul de căutare afișează legături relevante către pagini web care conțin ceva potrivit căutării.
sait sigur
Un sait care folosește criptare în conexiunile cu &brandShortName; pentru a împiedica alți utilizatori malițioși de pe internet să vadă datele transmise. Când vizitați saituri sigure, &brandShortName; afișează o iconiță reprezentând un lacăt în bara de stare și în bara adresei. &brandShortName; afișează și domeniul saitului în bara de stare (pentru a împiedica saiturile malițioase să fure datele introduse de dumneavoastră) și schimbă culoarea de fundal a barei de adrese din alb în galben.
Unele saituri pot conține și date sigure și nesigure; datele nesigure nu sunt în general private, deși nu se poate garanta că este așa. Pentru asemenea saituri &brandShortName; va afișa o iconiță reprezentând un lacăt tăiat în bara de stare și bara adresei și nu va afișa domeniul saitului în bara de stare pentru ca dumneavoastră să știți că pagina web curentă nu este complet sigură.
Secure Sockets Layer (SSL)
Un protocol care asigură autentificarea mutuală între un client și un server cu scopul de a stabili o conexiune autentificată și criptată. SSL rulează deasupra TCP/IP și sub HTTP, LDAP, IMAP, NNTP, și alte protocoale de rețea de nivel înalt. Noul standard Internet Engineering Task Force (IETF) numit Transport Layer Security (TLS) este de asemenea bazat pe SSL. Vedeți și autentificare, criptare.
server
Un program (cum ar fi unul care servește pagini web) care primește cereri de la un client și trimite informații către acesta. În mod normal, clientul se află pe un alt calculator decât serverul. Calculatorul pe care rulează programul server este și el numit server.
Bară de stare
Bara care apare în partea de jos a fiecărei ferestre &brandShortName;. Arată iconițe reprezentative pentru starea programului în dreapta.
TLS
Vedeți Secure Sockets Layer (SSL).
Transmission Control Protocol/Internet Protocol (TCP/IP)
Un protocol Unix folosit pentru a conecta calculatoare rulând diverse sisteme de operare. TCP/IP este un protocol esențial pentru internet și a devenit un standard global.
Uniform Resource Locator (URL)
Formatul standardizat al adresei unei pagini care ajută &brandShortName; să localizeze un fișier sau altă resursă pe web (de exemplu, http://www.mozilla.org). Tastați URL-uri în bara adresei din &brandShortName; pentru a accesa pagini web. URL-urile sunt folosite și ca legături în pagini web și ele duc către alte pagini. Sunt cunoscute și sub numele de adrese internet sau adrese web.
pagină web
Un singur document pe World Wide Web care este specificat de o adresă sau URL unică. O pagină web poate conține text, hiperlegături și grafică.
sait web
Un grup de pagini web legate prin hiperlegături și gestionate de o singură companie, organizație sau individ. Un sait web poate include text, grafică, fișiere audio și video și legături către alte saituri web.
World Wide Web
Pe scurt, web. O porțiune a internetului care este alcătuită din pagini web stocate pe mai multe servere web și care pot fi afișate în clienți numiți navigatoare web (browsere) (cum e &brandShortName;).

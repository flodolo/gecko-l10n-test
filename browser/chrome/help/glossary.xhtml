อภิธานศัพท์นี้เป็นการให้ข้อมูลข่าวสารเท่านั้น ไม่ได้หมายความว่าจะเชื่อได้ในความสมบูรณ์ หรือเชื่อถือในคำอธิบายศัพท์ที่ระบุไว้ด้านล่าง หรือความเป็นส่วนตัว และ/หรือ ความแตกต่างในระบบความปลอดภัยในเทคโนโลยีที่อธิบายไว้

อภิธานศัพท์

authentication
การใช้รหัสผ่าน, ใบรับรอง, เลขหมายยืนยันตนส่วนบุคคล (PIN), หรือ ข้อมูลข่าวสารอื่นที่เพื่อยืนยันระบุตัวตนผ่านเครื่อข่ายคอมพิวเตอร์
bookmark
หน้าเว็บที่จัดเก็บที่อยู่ (URL) ซึ่งทำให้สามารถไปยังหน้าเว็บนั้นได้ง่ายโดยการคลิกที่ไอคอนคั่นหน้าเว็บในแถบเครื่องมือที่คั่นหน้าเว็บหรือเลือกชื่อที่คั่นหน้าเว็บจากรายการที่คั่นหน้าเว็บ
แถบเครื่องมือที่คั่นหน้าเว็บ
แถบเครื่องมือกำหนดเองซึ่งปรากฏใต้แถบที่ตั้งโดยปริยายใน &brandShortName;ประกอบด้วยปุ่มสำหรับที่คั่นหน้าเว็บโปรด (หรือโฟลเดอร์ของกลุ่มที่คั่นหน้าเว็บ) ซึ่งสามารถเพิ่มเข้าหรือย้ายออกได้
cache
หน้าเว็บที่คัดลอกและเก็บสะสมไว้ในคอมพิวเตอร์ฮาร์ดดิสก์หรือในแรม (RAM) &brandShortName;การเพิ่มขึ้นหน้าเว็บคัดลอกเมื่อเรียกดู เมื่อคลิกลิงก์หรือพิมพ์ URL ไปยังหน้าเว็บที่ระบุ สำหรับแคชที่มีการคัดลอก &brandShortName;เปรียบเทียบกับแคชที่คัดลอกกับต้นฉบับ ถ้าไม่มีการเปลี่ยนแปลง &brandShortName; จะใช้แคชที่คัดลอกไว้มากกว่าไปยังหน้าต้นฉบับที่ระบุเพื่อประหยัดการประมวลผลและเวลาดาวน์โหลด
certificate
ข้อมูลดิจิทัลที่เทียบเท่ากับบัตรประชาชน ใบรับรองที่ระบุชื่อแต่ละบุคคล, บริษัทที่สังกัดอยู่ของเอกลักษณ์นั้นเมื่อข้อมูลดิจิทัลลงชื่อในข้อความหรือข้อมูลอื่น ลายเซ็นดิจิทัลสำหรับข้อความนั้นจะสร้างด้วยความเชื่อเหลือของกุญแจส่วนตัวซึ่งจะตอบสนองต่อกุญแจส่วนตัวในใบรับรองของคุณ
client
ซอฟต์แวร์ (เช่นตัวค้นหาอินเทอร์เน็ต) ที่ส่งคำร้องขอและรับข้อมูลข่าวสารจาก เซิร์ฟเวอร์ที่ทั่วไปทำงานบนคอมพิวเตอร์ที่แตกต่างกัน คอมพิวเตอร์เครื่องที่มีซอฟท์แวร์ลูกข่ายทำงานจะถูกระบุเป็นเครื่องลูกข่าย
cookie
ข้อมูลข่าวสารส่วนเล็กน้อยที่เก็บอยู่ในคอมพิวเตอร์ของบางเว็บไซต์ เมื่อเยี่ยมชมเว็บ เว็บจะถามถึง &brandShortName; เพื่อวางคุกี้หนึ่งหรือมากกว่าหนึ่งคุกกี้ในฮาร์ดดิสก์ เมื่อกลับมายังเว็บไซต์นั้นทีหลัง&brandShortName; จะส่งคุกกี้ที่เป็นของเว็บนั้นไป คุกกี้จะช่วยเว็บไซต์ติดตามข้อมูลข่าวสารของคุณ เช่น เนื้อหาในตะกร้าสินค้าคุณสามารถตั้งค่าตัวเลือกคุกกี้ในการควบคุมว่าคุกกี้จะถูกใช้แค่ไหนและข้อมูลข่าวสารแค่ไหนที่จะให้เว็บไซต์จัดเก็บไว้
cryptography
ศิลปะและการฝึกเข้าสัญญาณ(เข้ารหัสลับ) และการถอดสัญญาณ (ถอดรหัสลับ) ข้อมูลข่าวสาร ตัวอย่างเช่นเทคนิคการเข้ารหัสลับในการใช้เพื่อเข้าสัญญาณหรือถอดสัญญาณข้อมูลข่าวสารที่ไหลระหว่างเว็บไซต์การค้าและ &brandShortName;.
decryption
กระบวนการในการถอดสัญญาณข้อมูลที่เข้ารหัสลับไว้ ดูใน การเข้ารหัสลับ.

encryption
กระบวนการเข้าสัญญาณข้อมูลข่าวสารเป็นทางในการปลอมแปลงในตัวเอง เช่นการเข้ารหัสลับในการเชื่อมต่อระหว่างคอมพิวเตอร์ทำให้ยากมากต่อเว็บไซต์ที่สามจะถอดรหัสสัญญาณ หรือ ถอดรหัสลับ,ข้อมูลข่าวสารที่ไหลผ่านในการเชื่อมต่อ การเข้ารหัสลับข้อมูลข่าวสารสามารถถอดรหัสลับได้เฉพาะบางคนที่เป็นเจ้าของกุญแจที่เหมาะสม
Extensible Markup Language (XML)
มาตรฐานเปิดสำหรับรายละเอียด ไม่เหมือน HTML, XML อนุญาตให้นักพัฒนาหน้าเว็บในการจำแนกป้ายชื่อพิเศษข้อมูลข่าวสารเพิ่มเติม ดูในเอกสารออนไลน์ W3C Extensible Markup Language (XML).
feed
หน้าเว็บ XML ที่ประกอบด้วยรายการลิงก์ไปยังหน้าเว็บอื่นโปรแกรมพิเศษที่สามารถอ่านฟีดเพื่อสร้างรายการหัวเรื่องจากลิงก์ จะปรับปรุงให้ทันสมัยโดยอัตโนมัติเมื่อมีรายการเปลี่ยนแปลงเว็บไซต์ข่าวใช้ฟีดที่ตีพมพ์หัวเรื่องล่าสุดอย่างรวดเร็ว และ วารสารส่วนบุคคลที่ใช้ฟีดบ่อย ๆ ในการประกาศผู้เยี่ยมชมในข้อมูลที่เข้ามาใหม่ ดูใน ที่คั่นหน้าเว็บสด.
โพรโทคอลส่งไฟล์ (FTP)
มาตรฐานที่อนุญาตให้ผู้ใช้ส่งไฟล์จากคอมพิวเตอร์เครื่องหนึ่งไปยังอีกเครื่องผ่านเครื่อข่าย สามารถใช้&brandShortName; ในการรับไฟล์ใหม่ด้วยการใช้ FTP
FIPS PUBS 140-1
Federal Information Processing Standards Publications (FIPS PUBS) 140-1 is a US government standard for implementations of cryptographic modules — hardware or software that encrypts and decrypts data or performs other cryptographic operations (such as creating or verifying digital signatures). Many products sold to the US government must comply with one or more of the FIPS standards.
หน้าแรกเว็บไซต์
หน้า &brandShortName; ถูกตั้งให้แสดงทุกครั้งที่คุณเริ่มหรือเมื่อคลิกปุ่ม หน้าแรก ใช้เพื่ออ้างถึงหน้าหลักของเว็บไซต์ที่สามารถใช้สำรวจเว็บไซต์ที่เหลือได้
Hypertext Markup Language (HTML)
รูปแบบเอกสารที่ใช้สำหรับหน้าเว็บ มาตรฐาน HTMLจำแนกถึง ป้ายชื่อหรือรหัส, ใช้ในการจำแนกโครงเนื้อหา, ตัวอักษรรูปแบบลีลา, รูปและส่วนประกอบอื่นซึ่งรวมกันทำเป็นหน้าเว็บ
Internet
เครือข่ายทั่วโลกกว้างจากคอมพิวเตอร์นับล้านที่สื่อสารกันระหว่างแต่ละเครื่องใช้โพรโทคอลมาตรฐานเช่น TCP/IP. โดยกำเนิดขึ้นจากการพัฒนาทางทหารของสหรัฐอเมริกาเมื่อปี 1969 อินเทอร์เน็ตโตขึ้นโดยรวมถึงสถาบันการศึกษาและสถาบันวิจัยและปลายทศววรรษที่ 1990 ก็รวมธุรกิจนับล้าน องค์กร หรือเว็บไซต์ส่วนตัวเข้าไปด้วยปัจจุบันอินเทอร์เน็ตใช้ในจดหมายอิเล็กทรอนิกส์ ค้นหาWorld Wide Web เครื่องมือสื่อสารสำเร็จรูป รายการจดหมาย หรือกลุ่มผู้ใช้ และวัตถุประสงค์อื่นอีกมากมาย
ที่อยู่โพรโทคอลอินเทอร์เน็ต (IP address)
ที่อยู่คอมพิวเตอร์TCP/IP บนเครือข่ายคอมพิวเตอร์ทุกเครื่องบนอินเทอร์เน็ตจะมีที่อยู่ IP ลูกข่าย มีทั้งที่อยู่ IP ถาวรหรืออีกอันที่มอบให้เฉพาะ เมื่อเชื่อมต่ออินเทอร์เน็ตแต่ละครั้ง
Java
A programming language developed by Sun Microsystems. A single Java program can run on many different kinds of computers, thus avoiding the need for programmers to create a separate version of each program for each kind of computer. After Java is installed, &brandShortName; can automatically download and run Java programs (also called applets).
JavaScript
ต้นร่างภาษาที่ทั่วไปใช้ในการสร้างหน้าเว็บ นักทำโปรแกรมใช้ต้นร่างจาวาในการทำให้หน้าเว็บมีการสื่อสารระหว่างกันมากขึ้น ต้าร่างจาวาใช้บ่อยในรูปแบบการยืนยันตนหรือปุ่มเลือก ต้นร่างจาวายังสามารถใช้กับจาวา แต่เป็นภาษาที่แยกจากกันโดยเทคนิค จาวาไม่ต้องการต้นร่างจาวาเพื่อทำงานได้ถูกต้อง
ไลฟ์บุ๊กมาร์ก
ที่คั่นหน้าเว็บแบบที่พิเศษที่ประพฤติตัวเป็ยโฟลเดอร์ที่ประกอบด้วยลิงก์ในฟีด สามารถสร้างที่คั่นหน้าเว็บสดโดยการเยี่ยมเว็บไซต์ที่มีฟีดคลิกตรงไอคอนที่คั่นหน้าเว็บสดในแถบที่ตั้ง และเลือกฟีดที่ต้องการจะใช้

รหัสผ่านหลัก

Plugin
โปรแกรมเสริม (ปลั๊กอิน) เพิ่มความสามารถให้ &brandShortName; เช่น ความสามารถในการเล่นสื่อเสียงหรือวิดีโอคลิป. ไม่เหมือนกับโปรแกรมช่วยเหลือชนิดอื่น โปรแกรมเสริมจะติดตั้งตัวมันเองลงในไดเรกทอรีโปรแกรมเสริมภายในไดเรกทอรีหลักของ &brandShortName;.การติดตั้งไปยังปลายทางและสามารถเปิดได้ด้วย &brandShortName; ด้วยตัวเอง (ภายใน) ตัวอย่างเช่น โปรแกรมเสริมเสียงจะทำให้สามารถฟังไฟล์เสียงบนหน้าเว็บหรือข้อความจดหมายอิเล็กทรอนิกส์. ตัวอย่างของโปรแกรมเสริม เช่น Flash Player และ Java

เครื่องมือค้นหา
โปรแกรมที่ตั้งบนเว็บที่อนุญาตให้ผู้ใช้ค้นหาหรือกู้ข้อมูลข่าวสารที่เฉพาะจาก World Wide Web. เครื่องมือค้นหาอาจค้นหาข้อความเต็มของเอกสารเว็บหรือรายการคำหลัก ที่สามารถใช้โดยบรรณารักษ์ผู้ซึ่งตรวจสอบเอกสารเว็บและดัชนีด้วยมือ สำหรับการกู้ โดยทั่วไปผู้ใช้สามารถพิมพ์คำหรือวลีลงในกล่องค้นหาและเครื่องมือค้นหาจะแสดงลิงก์หน้าเว็บที่กู้ขึ้นมาให้
เว็บปลอดภัย
เว็บที่ใช้ การเข้ารหัสลับ ในการเชื่อมต่อกับ&brandShortName; เพื่อป้องกันผู้ใช้อินเทอร์เน็ตที่ประสงค์ร้ายในการชมข้อมูลที่ถ่ายทอดเมื่อเข้าชอมเว็บไซต์ที่ปลอดภัย &brandShortName; จะแสดงไอคอนใส่กุญแจที่ แถบสถานะและ แถบที่ตั้ง &brandShortName; ยังแสดงถึง ชื่อโดเมนของเว็บไซต์ในแถบสถานะ (เพื่อป้องกันจากเว็บไซต์ประสงค์ร้ายขโมยข้อมูล) และเปลี่ยนพื้นหลังของแถบที่ตั้งเป็นสีเหลือง
บางเว็บไซต์ประกอบด้วยข้อมูลที่ปลอดัยและไม่ปลอดภัย ข้อมูลที่ไม่ปลอดภัยส่วนใหญ่จะไม่เป็นส่วนตัว อย่างไรก็ตามไม่มีการรับรองว่าจะเป็นเช่นนั้นสำหรับ เว็บไซต์ &brandShortName; จะแสดงไอคอนเข้ากุญแจกับการตัดผ่านตัวไอคอนในแถบสถานะและแถบที่ตั้งและจะไม่แสดงชื่อโดเมนของเว็บไซต์ในแถบสถานะ ทำให่สามารถรู้ได้ว่าหน้าที่กำลังเข้าชมอยู่ขณะนี้ไม่ปลอดภัยทั้งหมด
Secure Sockets Layer (SSL)
โพรโทคอลที่อนุญาตให้การยืนยันตัวบุคคลซึ่งกันและกันระหว่างลูกข่าย and a เซิร์ฟเวอร์ สำหรับวัตถุประสงค์ในการแสดงการยืนยันตัวตนและเข้ารหัสลับในการเชื่อมต่อ SSL ทำงานบน TCP/IP และภายใต้HTTP, LDAP, IMAP, NNTP, และ เครือข่ายโพรโทคอลขั้นสูงแบบอื่น มาตรฐานภารกิจกำลังวิศวกรรมทางอินเทอร์เน็ตแบบใหม่ (IETF) เรียกว่า Transport Layer Security (TLS) อยู่บน SSL. ดู การยืนยันตน, การเข้ารหัสลับ.
server
ซอฟท์แวร์ (ซอฟท์แวร์ที่ทำงานให้หน้าเว็บนั้น) รับคำขอจากและส่งข้อมูลข่าวสารไปยังลูกข่าย ทำงานอยู่ให้ระบุเป็นเซิร์ฟเวอร์
แถบสถานะ
แถบเครื่องมือที่ปรากฏอยู่ใต้ &brandShortName; หน้าต่างจะแสดงไอคอนสถานะทางด้านซ้าย
TLS
ดู Secure Sockets Layer (SSL).
Transmission Control Protocol/Internet Protocol (TCP/IP)
โพรโทคอลยูนิกส์ที่ใช้เชื่อมต่อคอมพิวเตอร์ที่ทำงานระบบกระบวนการที่หลากหลายTCP/IP เป็นอินเทอร์เน็ตโพรโทคอลที่จำเป็นและกลายมาเป็นมาตรฐานโลก
Uniform Resource Locator (URL)
ที่อยู่มาตรฐานที่บอก &brandShortName; ว่าจะตั้งไฟล์ที่ไหนหรือแหล่งที่มาของเว็บอื่น(เช่น http://www.mozilla.org) พิมพ์ URLs ในแถบที่ตั้ง &brandShortName; เพื่อเข้าสู่หน้าเว็บURLs ยังใช้ในลิงก์ของหน้าเว็บเพื่อไปยังหน้าเว็บอื่น ซึ่งรู้จักทั่วไปใน อินเทอร์เน็ตหรือที่อยู่เว็บ
หน้าเว็บ
เอกสารเดี่ยวที่อยู่บน World Wide Web ที่ระบุที่อยู่หรือ URL เฉพาะหนึ่งเดียวหน้าเว็บประกอบด้วยเนื้อหา, ไฮเปอร์ลิงก์ และกราฟิก
เว็บไซต์
กลุ่มของหน้าเว็บลิงก์ที่มีความเกี่ยวข้องกันโดยไฮเปอร์ลิงก์และจัดการด้วยบริษัท องค์กร หรือบุคคลเพียงคนเดียวเว็บไซต์อาจประกอบด้วยเนื้อหา กราฟิก เสียง และไฟล์วิดีโอ และลิงก์ไปยังเว็บไซต์อื่น
World Wide Web
ส่วนใหญ่รู้จักในชื่อเว็บ บางส่วนของอินเทอร์เน็ตที่สร้างเป็นหน้าเว็บบันทึกโดยเว็บเซิร์ฟเวอร์ และแสดงโดย ลูกข่าย เรียกตัวค้นหาเว็บ (เช่น &brandShortName;).

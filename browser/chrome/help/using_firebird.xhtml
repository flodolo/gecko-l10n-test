使用 &brandFullName;

欢迎使用 &brandFullName;！&brandShortName; 是一款因特网浏览器，您可以用它访问网页或搜索网络。

本节内容包括：
  • 浏览网页
  • 搜索
  • 复制、保存并打印页面
  • 提高速度与效率

浏览网页

查看您的主页

当您启动 &brandShortName; 时，将看到您设置的主页。默认情况下，您看到的会是 &brandShortName; 的首页。

提示：

  • 使用全屏模式可以在屏幕上显示更多网页内容。在全屏模式下 &brandShortName; 的工具栏将会压缩为一个小工具条。要启用全屏模式，只需选择“查看”>“全屏”或按 F11。
  • 按 &altKey;+Home 可以快速前往您的主页。

转到其他页面

在地址栏输入因特网地址或 URL，您可以转到一个新的网页。URL 通常以 "http:// " 开头，后跟一个或多个用以识别地址的名称。例如，"http://www.mozilla.org/ "。

  1. 单击地址栏可以选择已访问过的 URL。
  2. 输入您想要访问的页面的 URL。您输入的 URL 将替换地址栏中已有的文本。
  3. 按下 &enterKey;。

提示：按 &accelKey;+L 可以快速选中地址栏的 URL。

不知道 URL？试试在地址栏中输入您要访问的网页的具体信息，例如它的名称，然后按 &enterKey;。这将使您转到 Google 关于该项目的前几项搜索结果。

单击链接

大多数网页都包含链接，您可以单击这些链接转到其他页面。

  1. 将鼠标指针移至链接上，直至变为手指指示图标。大多数链接都是带下划线的文本，但按钮或者图片也可以是一个链接。
  2. 单击链接。网络在定位链接的页面时，在窗口底部会显示状态信息。

返回历史页面

有以下几种重返页面的方法：

  • 要后退或前进到某页，可以单击后退或前进按钮。
  • 要后退或其前进多个页面，单击后退或前进按钮上的小三角形。您会看到最近访问页面的列表，您可以在列表中选择要返回的页面。
  • 要查看您在地址栏中输入过的所有 URL，单击地址栏右端的向下箭头。您可以在列表中选择想要查看的页面。
  • 要选择您在当前会话期间访问的页面，请打开“历史”菜单，然后使用该菜单底部的列表。
  • 要选择过去几个会话期间访问的页面，请打开历史菜单，然后选择在侧栏显示。您将看到历史侧栏。历史侧栏会显示一列文件夹，单击文件夹则显示子文件夹或网页的标题。要查看某页，可单击该页面的标题。

停止与重新载入

如果一个页面载入速度太慢或是您不再希望查看某一页，可以单击停止按钮。

要重新载入当前页面或想要获得页面的最新版本，单击重新载入按钮或按 &accelKey;+R。

标签式浏览

在您一次访问多个网页的时候，您可以使用标签式浏览，更迅速、更方便的浏览网络。

标签式浏览让您可以在单独一个 &brandShortName; 窗口中打开多个标签页，每个标签页显示一个网页。您不必为了访问几个不同的网页而打开多个窗口，这样可以为您节省桌面空间。您不需要切换到另一个窗口就可以方便的打开、关闭或者重新载入网页。

需要更多信息，参见标签式浏览。

使用侧栏

侧栏是位于屏幕左侧的一个区域，您可以用来查看书签或历史。扩展还可以增加新的侧栏使用方式。

要查看侧栏中的项目，选择“查看”>“侧栏”。在这里您可以选择您需要的“侧栏”标签。

搜索

搜索网络

要搜索关于某个特定主题的网页非常简单，只需要在 &brandShortName; 的搜索栏中输入几个关键词。

例如，如果您想查找有关“芭比娃娃”的信息：

  1. 在搜索栏中单击鼠标。
  2. 输入短语芭比娃娃。您输入的短语会覆盖当前在搜索栏中的所有文本。
  3. 按下 &enterKey; 开始搜索。

在 &brandShortName; 的窗口中将显示关于“芭比娃娃”的搜索结果。

选择搜索引擎

您可以切换搜索引擎。单击搜索引擎图标，然后在所提供的搜索引擎中进行选择。有些搜索引擎，例如 Google，会搜索整个网络；而另外一些，像 Amazon.com，仅搜索特定站点。

管理搜索引擎

单击搜索引擎图标，选择管理搜索引擎...可以添加、重新排列、删除或恢复默认搜索引擎。选择一个搜索引擎，单击相应按钮可以在列表中移动或删除该搜索引擎。通过单击获得更多搜索引擎...链接可以安装新的搜索引擎。

在网络中搜索网页上选定的关键词

&brandShortName; 允许您在网络中搜索您在网页上选定的关键词：

  1. 在网页中选择（高亮显示）某个词语。
  2. 右键单击按 &ctrlKey;，并单击鼠标键，然后在弹出菜单中选择通过 [搜索引擎] 搜索“[您选择的词语]”。

&brandShortName; 将会打开一个新的标签页，使用当前选择的搜索引擎搜索您选择的关键词。

在页面中搜索

在您用 &brandShortName; 查看的当前页面中查找文本：

  1. 按 &accelKey;+F 或选择“编辑”>“在当前页查找...” ，将在 &brandShortName; 底部打开查找工具栏。
  2. 输入您要查找的文本。您在查找框中输入文本的同时，搜索会自动开始。
  3. 查找工具栏提供以下选项：
    • 下一个：查找页面中当前光标位置后面的文本。
    • 上一个：查找当前光标位置前面的文本。
    • 全部高亮显示：高亮显示您在当前页面中搜索到的字符串。
    • 大小写匹配：仅搜索与您搜索的字符串大小写相同的文本。

要再次搜索相同的单词或短语，按 F3 或选择“编辑”>“查找下一个”。

提示：在 &pref.menuPath; 中选择高级面板的常规标签，然后选择输入时即时查找文本 &pref.singular;，可启用查找工具栏的快速查找模式。当该模式启用时，快速查找工具栏会在您一开始输入时自动打开并开始搜索。和查找工具栏不一样，保持几秒钟的非活动状态后，快速查找工具栏会自动关闭。

复制、保存并打印页面

复制部分页面

从页面复制文本：

  1. 选择文本。
  2. 在菜单栏选择“编辑”>“复制”。

您可以将文本粘贴到其他程序中。

从页面复制链接 (URL) 或图片链接：

  1. 将鼠标指针移至链接或图片上面。
  2. 右键单击按 &ctrlKey; 并单击链接或图片，显示弹出菜单。
  3. 选择复制链接地址或复制图片地址。如果图片同时也是一个链接，那么这两个菜单项都可以选择。

您可以将链接粘贴到其他程序中，或者粘贴到 &brandShortName; 的地址栏。

保存全部或部分页面

在 &brandShortName; 中保存整个页面：

  1. 选择 “文件”>“页面另存为...”。您会看到“另存为”对话框。
  2. 选择页面的保存位置。
  3. 选择您要保存页面的格式：
    • 网页，全部：连同图片一起，保存整个网页。此选项让您可以像显示包含图片的原始页面一样查看保存的页面，但它可能无法保持原始页面的 HTML 链接结构。&brandShortName; 将在页面保存位置创建一个新的目录用以保存图片和显示整个网页所需的其他文件。
    • 网页，仅 HTML：保存原始页面，但不包括图片。此选项将保持文件原始的 HTML 链接结构。
    • 文本文件：将原始页面保存为文本文件。此选项不会保留原始的 HTML 链接结构，但可以让您用任意一个文本编辑器查看文本格式的网页。
  4. 为页面输入一个文件名称，然后单击保存。

保存网页内部的帧：

  1. 将鼠标指针置于帧的内部。
  2. 按下 &ctrlKey; 并单击右键单击帧，显示一个弹出菜单。
  3. 在子菜单中选择“本帧”>“帧另存为”。您将看到“另存为”对话框。
  4. 选择页面的保存位置。
  5. 选择您要保存页面的格式。
  6. 为页面输入一个文件名，然后单击保存。

将文件保存到您的硬盘中，能够让您离线的时候也可以查看该页面。

从页面保存图片：

  1. 将鼠标置于图片上面。
  2. 右键单击按 &ctrlKey; 并单击图片，显示一个弹出菜单。
  3. 选择图片另存为。您将看到“保存图片”对话框。
  4. 选择图片的保存位置。
  5. 为图片输入一个文件名，然后单击保存。

不显示而直接保存页面（用于保存不准备查看的非格式页面）：

  1. 将鼠标指针置于指向页面的链接上。
  2. 右键单击按 &ctrlKey; 并单击链接，显示一个弹出菜单。
  3. 选择链接另存为。您将看到“另存为”对话框。
  4. 选择页面的保存位置。
  5. 为页面输入一个文件名，然后单击保存。

重要信息：某些链接在您单击后会自动下载文件并保存到您的硬盘当中。这种链接的 URL 常常以 "ftp" 开头或以文件类型扩展名结束，例如 "au" 或 "mpeg"。这些链接可能会传输软件、音频、电影文件并且可以启动支持这些文件的辅助应用程序。

提示：要把一幅图片设置为您的桌面背景，可以在图片上单击右键按下 &ctrlKey; 并单击鼠标键，然后在弹出菜单中选择设为桌面背景...。

打印页面

打印当前页面：

  • 选择“文件”>“打印”。

打印选定文本：

  • 在当前页面选择文本。
  • 选择“文件”>“打印”。出现打印对话框。
  • 在打印范围下面，选择选定范围。

网页作者和打印页面的尺寸，而不是屏幕上窗口的尺寸，决定了打印页面的内容布局。为适应纸张尺寸，文本将自动换行，并重新布置图片。

使用打印预览

您可以使用“打印预览”在打印页面之前查看页面如何显示：

  • 选择“文件”>“打印预览”。

在“打印预览”中，您可以对您要打印的页面进行以下操作：

  • 查看每一页的预览：单击“下一页”、“上一页”、“第一页”或“最后一页” 按钮，可以在页面之间翻动。
  • 更改页面比例（尺寸）：在比例下拉菜单中，选择 "50%" ，让页面以原始页面的一半尺寸显示。要自动调整页面大小以适应页面宽度，选择“调整到适合”。您也可以选择“定制...”，输入您自己设定的百分比。
  • 更改页面方向：选择竖向 将以短边向上布置页面，选择横向 将以长边向上侧向布置页面。
  • 进行页面设置：单击页面设置进一步定制您要打印的页面。
  • 开始打印：单击打印开始打印页面。

注意：某些打印预览的功能在 Mac OS 和 Linux 上可能有些不同或无法使用。

使用页面设置

注意：某些页面设置的功能在 Mac OS 和 Linux 上可能有些不同或无法使用。

您可以使用“页面设置”来定制在 &brandShortName; 中如何打印页面：

  • 选择“文件”>“页面设置”。

在“页面设置”中，您可以更改您要打印的页面的以下设置：

  • 格式与选项：选择方向、比例和其他选项：
    • 方向：
      • 竖向：选择此项 将以常规的短边向上布置页面。
      • 横向：选择此项 将以长边向上侧向布置页面。
    • 比例：输入原始页面的百分比。例如，输入 "50" 让每页尺寸为原始页面的一半。
      • 缩放以适应页面宽度：选择此项可以自动调整页面大小以适应页面宽度。
    • 选项:
      • 打印背景（颜色与图片）：选择此项将打印背景图片和颜色。如果未选此项，则只打印前景（最前面的）图片和颜色。
  • 页边距与页眉/页脚：单击此标签设置页边距、页眉与页脚：
    • 页边距：
      • 顶部、底部、左边、右边：为顶部、底部、左、右各边输入指定单位的边距大小。
    • 页眉与页脚：每一个下拉列表表示一个页眉或页脚的区域。上面一行是页眉的左、中、右区域，下面一行是页脚的左、中、右区域。每个下拉列表中，可以选择以下选项：
      • --空白--：在该区域什么也不显示。
      • 标题：显示网页标题
      • URL：显示网页 URL（URL 常以 "http:// " 开头）。
      • 日期/时间：显示网页打印的日期和时间。
      • 第 # 页：显示每个页面的页码。
      • 第 # 页，共 # 页：显示页码以及页面总数。例如，如果您打印一份共 5 页的网页，在第 3 页将显示“第 3 页，共 5 页”。
      • 自定义...：您可以自己输入文本，打印特定信息。文本中可以包括以下代码：
        • &PT：页码及总数（比如：“第 3 页，共 5 页”）
        • &P：页码
        • &D：日期
        • &U：URL
        • &T：页面标题

提示：可以使用 打印预览查看更改页面设置之后的打印页面。

提高速度与效率

管理不同文件类型

&brandShortName; 可以处理很多类型的文件。但是，对于某些像电影或音乐这样的文件，&brandShortName; 需要能够处理这些文件的插件或外部应用程序。如果没有需要的插件或者不知道使用什么应用程序来打开文件，&brandShortName; 仍然会将文件保存到您的硬盘当中。在保存文件的时候，您可以利用下载管理器时刻掌握文件的下载情况。

插件在 &brandShortName; 内部运行，它可以扩展 &brandShortName; 的功能。像 Sun Java、Macromedia Flash 以及 RealNetworks RealPlayer 等等，这些插件允许 &brandShortName; 显示多媒体文件，运行电影、动画、游戏等小应用程序。您可以在 Mozilla Update 的 插件页面上找到常用插件。

当文件无法在 &brandShortName; 内部打开时，您可以选择在 &brandShortName; 外部启动应用程序打开文件。例如，可以用您选择的媒体播放器来播放 MP3 文件。

在下载动作对话框里，您可以指定 &brandShortName; 如何处理它不能显示的文件。要打开该对话框，选择 &pref.menuPath;，打开内容面板，然后单击文件类型部分的管理...按钮。您可以更换或是移除针对某个文件类型的动作。在文件类型中，可以获得关于下载动作对话框的更多信息。

当您单击一个文件时，如果 &brandShortName; 不知道如何处理该文件，将显示打开对话框，提供以下选项：

  • 打开方式：&brandShortName; 使用系统设置的默认程序打开文件。您也可以在下拉列表中选择其他...使用不同的程序。
  • 保存至磁盘：&brandShortName; 允许您选择文件的保存位置。如果您已经在 &pref.pluralCaps; 的主要面板中选择了保存文件至（此文件夹），那么文件将自动保存到默认文件夹；否则 &brandShortName; 会询问您保存文件的位置。
  • 以后自动采用相同的动作处理此类文件：如果选择此项，在您下次下载此类文件时，&brandShortName; 会自动执行选择的动作。

提示：您可以在 下载动作对话框中清除您保存的对于以后自动采用相同的动作处理此类文件的 &pref.plural;。

将 &brandShortName; 设置为默认浏览器

&brandShortName; 可以显示因特网中以及保存在您硬盘中的网页。为了方便地打开网页，您可以将 &brandShortName; 设置为您的默认浏览器。

要将 &brandShortName; 设置为默认浏览器，选择 &pref.menuPath;，然后单击主要面板中的立即检查按钮。

更改缓存设置

&brandShortName; 将经常访问的页面的副本储存在缓存中。这样，在您查看该网页的时候，&brandShortName; 就不必每次都从网络上重新获取页面。

设置缓存大小：

  1. 选择 &pref.menuPath;。
  2. 打开高级面板，单击网络标签。
  3. 在最多使用一栏输入数字，指定缓存大小。默认大小为 50MB。

重要：缓存越大，可以快速取回的数据就越多，但也需要使用更多的硬盘空间。

当您退出 &brandShortName; 时，它要进行缓存维护。如果维护时间比您预期的长很多，可以尝试减小磁盘缓存。

在任何时间您都可以单击 &brandShortName; 导航工具栏上的重新载入按钮，重新载入页面。&brandShortName; 将载入并显示页面的最新版本。

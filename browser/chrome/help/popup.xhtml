Controlar las ventana emergentes

Este documento explica todas las &pref.plural; disponibles en &brandFullName; para controlar las ventanas emergentes.

En esta sección:
  • ¿Qué son las ventanas emergentes?
  • &pref.pluralCaps; del bloqueador de ventanas emergentes

¿Qué son las ventanas emergentes?

Las ventanas emergentes, o popups, son ventanas que aparecen automáticamente sin nuestro permiso. Pueden tener distintos tamaños pero normalmente no cubren toda la pantalla. Algunas ventanas emergentes se abren sobre la ventana actual de &brandShortName;, mientras que otras aparecen por debajo de la de &brandShortName; (popunders).

&brandShortName; permite controlar tanto popups como popunders mediante el panel de contenido de las &pref.pluralCaps;. El bloqueo de ventanas emergentes está activado por omisión, por lo que no hay que preocuparse de activarlo para evitar que aparezcan ventanas emergentes en &brandShortName;.

Al bloquear una ventana emergente, &brandShortName; muestra una barra de información, así como un icono en la barra de estado. Al pulsar ya sea el botón de &pref.pluralCaps; en la barra de información o el icono en la barra de estado, se muestra un menú con las siguientes opciones:

  • Permitir/bloquear ventanas emergentes para este sitio
  • Editar las &pref.pluralCaps; del bloqueador de ventanas emergentes
  • No mostrar el mensaje de información al bloquear ventanas emergentes
  • (mostrar una ventana emergente bloqueada)

Bloquear ventanas emergentes puede interferir con algunos sitios web: algunos sitios web, incluyendo algunos bancos, usan ventanas emergentes para características importantes. Bloquear todas las ventanas emergentes desactiva estas características. Para permitir que sitios web específicos usen ventanas emergentes a la vez que se bloquea a los demás, se pueden añadir sitios web específicos a la lista de sitios permitidos.

El bloqueo de ventanas emergentes no siempre funciona: aunque &brandShortName; bloquea la mayoría de las ventanas emergentes, algunos sitios web pueden mostrar ventanas emergentes usando métodos que no están cubiertos, incluso estando activado el bloqueo.

&pref.pluralCaps; del bloqueador de ventanas emergentes

Las &pref.pluralCaps; del bloqueador de ventanas emergentes están situadas en el panel Contenido de &pref.menuPath;.

Desde ahí se puede hacer lo siguiente:

  • Bloquear ventanas emergentes: desmarque esta &pref.singular; para desactivar por completo el bloqueo de ventanas emergentes.
  • Diálogo de excepciones: es una lista de sitios a los que se dese permitir mostrar ventanas emergentes. El diálogo tiene las siguientes opciones:
    • Permitir: púlselo para permitir añadir un sitio web a la lista de excepciones.
    • Borrar sitio: púlselo para borrar un sitio web de la lista de excepciones.
    • Borrar todos los sitios: púlselo para borrar todos los sitios web de la lista de excepciones.

Nota: el bloqueo de ventanas emergentes puede no funcionar siempre e interferir con algunos sitios web. Para más información sobre el bloqueo de ventanas emergentes, vea Qué son las ventanas emergentes.

Controlul ferestrelor popup

Acest document explică toate &pref.plural;le disponibile în &brandFullName; pentru controlul ferestrelor popup.

În această secțiune:
  • Ce sunt popup-urile?
  • &pref.pluralCaps;le de blocare a popup-urilor

Ce sunt popup-urile?

Ferestrele popup, sau popup-urile sunt ferestre care se deschid automat, fără permisiunea dumneavoastră. Mărimea lor variază, dar în mod normal nu acoperă tot ecranul. Unele popup-uri sunt deschise deasupra ferestrei curente &brandShortName;, iar altele sunt deschise în spatele ferestrei &brandShortName; (subpopup-uri).

&brandShortName; vă permite să controlați atât ferestrele popup care se deschid în spate, cât și cele care se deschid în față în panoul Conținut din &pref.pluralCaps;. Blocarea ferestrelor popup este activată implicit, deci nu trebuie să vă faceți griji legate de activarea acestei funcționalități pentru a preveni ferestrele popup să apară în &brandShortName;.

Când o fereastră popup este blocată, &brandShortName; afișează o bară de informații, similar cu această iconiță în bara de stare. Fie că apăsați butonul &pref.pluralCaps; din bara de informații sau icoana din bara de stare, un meniu este afișat cu următoarele opțiuni:

  • Permite/Blochează ferestre popup de la acest sait
  • Editează &pref.pluralCaps;le de blocare a ferestrelor popup...
  • Nu afișa mesaj informativ când sunt blocate ferestre popup
  • (arată o fereastră popup blocată)

Blocarea ferestrelor popup poate interfera cu funcționarea unor saituri web: Unele saituri web, inclusiv unele saituri ale unor bănci, folosesc ferestre popup pentru funcționalități importante. Blocarea tuturor ferestrelor popup dezactivează asemenea funcționalități. Pentru a permite unor saituri să folosească ferestre popup, în timp ce altele să fie blocate, puteți adăuga saituri web la lista de saituri cărora le este permis să afișeze popup-uri

Blocarea ferestrelor popup nu funcționează întotdeauna: Deși &brandShortName; blochează majoritatea ferestrelor popup, unele saituri web ar putea deschide ferestre popup folosind metode ascunse, chiar când acestea sunt blocate.

&pref.pluralCaps;le de blocare a popup-urilor

&pref.pluralCaps;le de blocare a ferestrelor popup se află în panoul Conținut din &pref.menuPath;.

De acolo, puteți face următoarele lucruri:

  • Blochează ferestrele popup: Nu bifați &pref.singular; aceasta pentru a dezactiva blocarea ferestrelor popup.
  • dialogul Excepții: Aceasta conține o listă de saituri cărora le permiteți să deschidă ferestre popup. Acest dialog are următoarele opțiuni:
    • Permite: Clic pentru a adăuga un sait web la lista de excepții.
    • Elimină sait: Clic pentru a elimina un sait web din lista de excepții.
    • Elimină toate saiturile: Clic pentru a elimina toate saiturile web din lista de excepții.

Notă: Blocarea ferestrelor popup nu funcționează întotdeauna și poate interfera cu buna funcționalitate a unor saituri web. Pentru mai multe informații, vedeți și Ce sunt popup-urile.

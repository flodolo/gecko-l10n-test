การควบคุมป็อปอัพ

เอกสารนี้อธิบาย&pref.plural;ที่มีอยู่ทั้งหมดใน &brandFullName;ที่ใช้สำหรับการควบคุมป็อปอัพ

ในส่วนนี้ :
  • ป็อปอัพคืออะไร
  • ตัวปิดกั้นป็อปอัพ &pref.pluralCaps;

ป็อปอัพคืออะไร

หน้าต่างป็อปอัพ หรือ ป็อปอัพ คือหน้าต่างที่ปรากฏออกมาอัตโนมัติโดยไม่ได้รับอนุญาตจากคุณขนาดของหน้าต่างจะแตกต่างกัน แต่ปกติแล้วจะไม่กินที่ทั้งหน้าจอบางป็อปอ้พจะลอยอยู่เหนือหน้าต่างปัจจุบันของ &brandShortName; ในขณะที่บางอันจะปรากฏอยู่ข้างใต้ (ป็อปอันเดอร์)

&brandShortName; อนุญาตให้คุณควมคุมทั้งป็อปอัพและป็อปอันเดอร์ทางตัวเลือกปรับแต่งเนื้อหา ใน&pref.pluralCaps; ตัวปิดกั้นป็อปอัพจะเปิดใช้งานโดยปริยาย ดังนั้นคุณไม่จำเป็นต้องกังวลเกี่ยวกับการเปิดใช้งานมันเพื่อป้องกันการแสดงป็อปอัพใน &brandShortName;

เมื่อปิดกั้นป็อปอัพ &brandShortName; จะแสดงแถบข้อมูล พร้อมกับไอคอนบนแถบสถานะ เมื่อคลิกที่ปุ่ม &pref.pluralCaps; ในแถบข้อมูลหรือไอคอนในแถบสถานะจะมีเมนูแสดงให้เลือกตัวเลือกต่อไปนี้ :

  • อนุญาต/ปิดกั้น ป็อปอัพสำหรับไซต์นี้
  • แก้ไข ตัวปิดกั้นป็อปอัพ &pref.pluralCaps;…
  • ไม่แสดงข้อความนี้ (ข้อความแจ้งข้อมูล) เมื่อป็อปอัพถูกปิดกั้น
  • (แสดงป็อปอัพที่ถูกปิดกั้น)

ตัวปิดกั้นป็อปอัพอาจรบกวนการทำงานของบางเว็บไซต์: บางเว็บไซต์รวมถึงเว็บไซต์ธนาคาร จะใช้ป็อปอัพสำหรับคุณสมบัติที่สำคัญการปิดกั้นป็อปอัพทั้งหมดจะปิดการใช้งานคุณสมบัติเหล่านั้น เพื่อที่จะอนุญาตให้บางเว็บไซต์ที่ระบุใช้ป็อปอัพได้ ในขณะที่ปิดกั้นที่เหลือทั้งหมด คุณสามารถเพิ่มเว็บไซต์เหล่านั้นเข้าไปในรายการของไซต์ที่อนุญาต

ตัวปิดกั้นป็อปอัพอาจจะทำงานไม่ได้ผลเสมอไป: ถึงแม้ว่า &brandShortName ปิดกั้นป็อปอัพส่วนใหญ่แต่บางเว็บไซต์อาจแสดงป็อปอัพได้แม้จะถูกปิดกั้นไว้โดยใช้วิธีที่ครอบคลุมไม่ถึง

ตัวปิดกั้นป็อปอัพ &pref.pluralCaps;

ตัวปิดกั้นป็อปอัพ&pref.pluralCaps;อยู่ในตัวเลือกปรับแต่งเนื้อหา ของ &pref.menuPath

จากที่นั่น คุณสามารถทำสิ่งต่อไปนี้ :

  • ปิดกั้นหน้าต่างป็อปอัพ: ไม่เลือก&pref.singular;นี้เพื่อปิดการใช้งานตัวปิดกั้นป็อปอัพทั้งหมด
  • กล่องโต้ตอบข้อยกเว้น: This is a list of sites that you want to allow to display pop-ups. The dialog has the following choices:
    • อนุญาต: คลิกอันนี้เพื่อเพิ่มเว็บไซต์เข้าไปยังรายการข้อยกเว้น
    • ลบไซต์: คลิกอันนี้เพื่อลบเว็บไซต์จากรายการข้อยกเว้น
    • ลบไซต์ทั้งหมด: คลิกอันนี้เพื่อลบเว็บไซต์ทั้งหมดในรายการข้อยกเว้น

หมายเหตุ: การปิดกั้นป็อปอัพอาจจะไม่ได้ผลเสมอไป และอาจรบกวนบางเว็บไซต์ สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการปิดกั้นป็อปอัพ ดูที่ ป็อปอัพคืออะไร

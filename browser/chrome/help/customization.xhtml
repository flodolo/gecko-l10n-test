Personalizando o &brandFullName;

Nesta seção:
  • Barras de ferramentas
  • Complementos (extensões e temas)

Barras de ferramentas

Este tópico ensina como reorganizar, ocultar, adicionar e remover barras de ferramentas no &brandShortName;. Para aqueles que não estão familiarizados com a terminologia, uma barra de ferramentas é a parte (normalmente cinza) localizada acima da área onde o site é exibido.

  • Barra de menus: é a barra de ferramentas que contém os menus do navegador (Arquivo, Editar, Ajuda, etc.).
  • Barra de navegação: é a barra que contém o Campo de endereço e os botões usados para a navegação nos sites.
  • Barra de favoritos: é a barra que contém os favoritos da pasta Barra dos favoritos.

Existem duas formas básicas de personalizar as barras de ferramentas no &brandShortName;:

  • Reorganizando itens das barras de ferramentas
  • Ocultando, Adicionando e Removendo barras de ferramentas

Reorganizando itens das barras de ferramentas

Para reorganizar os itens das suas barras de ferramentas (ícones, botões, caixas de texto, etc.), clique com o botão direitopressione &ctrlKey; e clique em qualquer parte das barras de ferramentas que não seja uma caixa de texto e clique em Personalizar…. Isso fará com que a janela de personalização de barras de ferramentas apareça.

Arraste os itens que quiser para dentro ou para fora de uma das barras de ferramentas (com exceção dos menus da barra de menus). Você também pode reorganizar os ícones que já estiverem em alguma das barras de ferramentas arrastando-os para a posição que desejar. Quando as barras de ferramentas e seus itens já estiverem da maneira que você desejar, clique em Fechar.

Itens especiais

Alguns itens especiais estão disponíveis ao personalizar as barras de ferramentas. Eles são os seguintes:

  • Separadores: permitem que você separe itens nas barras de ferramentas com uma pequena linha vertical.
  • Espaços: permitem que você separe itens nas barras de ferramentas com um espaço de tamanho fixo, normalmente por volta dos 20 pixels na maioria dos temas.
  • Espaços flexíveis: diferentemente dos espaços simples, esses espaços são dimensionados automaticamente para ocupar todo o espaço em branco das barras de ferramentas. Isso é útil para posicionar itens no final das barras de ferramentas.

Ocultando, Adicionando e Removendo barras de ferramentas

Você pode ocultar a barra de navegação e a barra de favoritos se você não quiser mantê-las na tela. Você não poderá remover a Barra de menus porque você não pode mover seus itens (Arquivo, Editar, Ajuda, etc.) para outra barra de ferramentas, e eles deverão estar sempre na tela.

Para exibir ou ocultar uma barra de ferramentas, clique em Exibir > Barras de ferramentas e clique no nome da barra de ferramentas que deseja exibir ou ocultar.

Para adicionar uma barra de ferramentas, clique em Exibir > Barras de ferramentas > Personalizar. Clique em Nova barra (na base da janela). Aparecerá um diálogo pedindo o nome da nova barra de ferramentas. Se você digitar um nome e clicar em OK a nova barra de ferramentas aparecerá. Agorá você poderá arrastar novos itens até ela. Para remover uma barra de ferramentas pessoal, arraste todos os seus itens para fora da barra e clique em Fechar. Você também pode exibir ou ocultar uma barra de ferramentas pessoal utilizando o procedimento descrito no parágrafo acima.

Complementos (extensões e temas)

Complementos são pequenos pacotes que alteram ou adicionam diferentes aparências ou funcionalidades ao &brandShortName;. Há dois tipos de complementos: extensões e temas.

Extensões
Extensões adicionam novas funcionalidades ao &brandShortName;. Elas podem adicionar qualquer coisa, desde um botão na barra de ferramentas até uma função completamente nova. Elas permitem que o &brandShortName; seja personalizado para atender as necessidades pessoais de cada usuário.
Temas
Temas modificam a aparência do &brandShortName;. Eles permitem que você modifique o visual e a sensação do &brandShortName; e personalize-o de acordo com o seu gosto pessoal. Um tema pode simplesmente mudar imagens de botões ou alterar cada parte da aparência do &brandShortName;.

Obtendo e Instalando complementos

Você pode baixar e instalar vários complementos a partir do site Mozilla Add-ons. Pesquisas na Internet permitirão encontrar muitos outros complementos.

Quando você clica em um link para instalar um complemento, o &brandShortName; exibe um diálogo pedindo permissão para instalar o complemento. Você poderá decidir entre baixar e instalar o complemento ou cancelar o processo

Usando o Gerenciador de Complementos

Complementos são organizados no Gerenciador de Complementos. Clique em Ferramentas > Complementos para abri-lo.

Atualizando complementos

Por padrão, o &brandShortName; verificará periodicamente a existência de atualizações para os seus complementos instalados. Se houver alguma atualização disponível, o &brandShortName; perguntará se você deseja instalar as atualizações na próxima vez que for iniciado. Você pode alterar esse comportamento na aba Atualizações do painel Avançado da janela &pref.pluralCaps;.

Você pode verificar a existência de atualizações manualmente selecionando o painel Extensões ou Temas e clicando no botão Localizar atualizações. Se houver atualizações disponíveis, o gerenciador exibirá o painel Atualizações, onde você poderá escolher as atualizações que deseja instalar. Clique no botão Instalar atualizações para atualizar os complementos selecionados. Será necessário reiniciar o &brandShortName; para que as alterações tenham efeito.

Desativando, Ativando e Desinstalando complementos

Se você estiver insatisfeito com algum complemento, você poderá desativá-lo temporariamente - caso você queira ativá-lo novamente no futuro - ou desinstalá-lo. Temas somente podem ser desinstalados, uma vez que todos os temas, exceto o tema exibido, são desativados automaticamente.

Para desativar ou desinstalar uma extensão, selecione a extensão e clique no seu botão Desativar ou Ativar. Para remover um complemento do &brandShortName;, selecione o complemento que deseja desinstalar e clique no seu botão Desinstalar. Será necessário reiniciar o &brandShortName; para que as alterações tenham efeito.

Informações adicionais

Informações adicionais sobre os complementos, como Visitar o site e Sobre, estão disponíveis selecionando o complemento desejado, pressionando &ctrlKey; e clicandoclicando com o botão direito no complemento e selecionando o respectivo item no menu de contexto.

Rotinas exclusivas de extensões e temas

Configurando extensões

Extensões normalmente fornecem um diálogo para sua configuração. Para abrir o diálogo de &pref.plural; de uma extensão, clique no painel Extensões, selecione a extensão desejada e clique no seu botão &pref.pluralCaps;, ou simplesmente dê um clique duplo sobre a extensão.

Trocando de tema

Para selecionar algum entre os temas instalados, clique no painel Temas, selecione o tema desejado e clique no seu botão Usar tema. Será necessário reiniciar o &brandShortName; para que as alterações tenham efeito.

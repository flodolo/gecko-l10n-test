Barras de herramientas

Aquí se explica cómo modificar, ocultar, añadir y borrar barras de herramientas en &brandShortName;. Para aquellos que no estén familiarizados con la terminología, una barra de herramientas es la caja (normalmente gris) que está encima del área que muestra la página web.

  • Barra de menú: es la barra de herramientas que contiene los menús del navegador (Archivo, Editar, Ayuda, etc.).
  • Barra de navegación: es la barra de herramientas que contiene la barra de direcciones y los botones usados para navegar por los sitios web.
  • Barra de herramientas de marcadores: es la barra de herramientas que contiene los marcadores de la carpeta de la barra de herramientas de marcadores.

Hay dos formas básicas de personalizar las barras de herramientas en &brandShortName;:

  • Modificar elementos en las barras de herramientas
  • Esconder, añadir y borrar barras de herramientas

Modificar elementos en las barras de herramientas

Para modificar los elementos (iconos, botones, cuadros de texto, etc.) en la barra de herramientas, pulse con el botón derechopulse &ctrlKey; y haga clic en cualquier parte de la barra de herramientas que no sea un recuadro de texto y seleccione Personalizarâ€¦. Esto abrirá el cuadro de diálogo para personalizar la barra de herramientas.

Pulse y arrastre cualquier elemento que se desee a o desde las barras de herramientas (excepto los elementos de menú). También se puede cambiar el orden de los iconos que ya estén en las barras de herramientas arrastrándolos y soltándolos donde se desee. Cuando las barras de herramientas y los elementos estén como se desee, pulse Terminado.

Elementos especiales de la barra de herramientas

Hay unos pocos elementos especiales que están disponibles cuando se personaliza la barra de herramientas. Son los siguientes:

  • Separadores: estos permiten separar elementos en las barras de herramientas con una pequeña línea vertical.
  • Espaciadores: estos permiten separar elementos en las barras de herramientas con un espacio fijo, normalmente unos 20 píxeles en la mayoría de los temas.
  • Espaciadores flexibles: los espaciadores le permiten separar elementos en las barras de herramientas con un espacio. Estos espaciadores normalmente ajustan su tamaño automáticamente para llenar todo el espacio vacío en la barra de herramientas. Son buenos para posicionar elementos en la parte derecha de una barra de herramientas.

Esconder, añadir y borrar barras de herramientas

Se puede esconder la barra de navegació y la barra de favoritos si no se quieren mantener en la pantalla. No se puede esconder la barra de menú porque no se pueden mover los elementos del menú (Archivo, Edición, Ayuda, etc.) a otra barra de herramientas, por lo que deben permanecer en la pantalla siempre.

Para mostrar o esconder una barra de herramientas, seleccione Ver > Barras de herramientas y haga clic en el nombre de la barra de herramientas que quiere mostrar o esconder.

Para añadir una barra de herramientas personalizada, seleccione Ver > Barras de herramientas > Personalizar. Haga clic en Añadir barra de herramientas (en la parte inferior del diálogo), lo que presentará un diálgo para intruducir un nombre. Si introduce un nombre y presiona Aceptar, la nueva barra de herramientas aparecerá. Ahora se pueden arrastrar y soltar elementos en ella. Para borrar una barra de herramientas personalizada, arrastre y suelte todos los elementos de ella y haga clic en Terminado. También se puede esconder una barra de herramientas personalizada tal como se describe en el párrafo anterior.

Complementos

Los complementos son pequeñas piezas de software que cambian o añaden elementos a la apariencia o funcionalidad de &brandShortName;. Hay dos tipos de complementos: extensiones y temas.

Extensiones
Las extensiones añaden nueva funcionalidad a &brandShortName;. Pueden añadir cualquier cosa, desde un botón en la barra de herramientas a una característica completamente nueva. Permiten a la aplicación ser personalizada para ajustarse a las necesidades personales de cada usuario a la vez que minimizan el tamaño y apariencia de la aplicación en sí misma.
Temas
Los temas modifican la apariencia de &brandShortName;. Le permiten cambiar el aspecto y comportamiento de &brandShortName; y personalizarlo a su gusto. Un tema puede cambiar simplemente las imágenes de los botones, o puede cambiar cada elemento de la interfaz de &brandShortName;.

Obtener e instalar complementos

Puede descargar e instalar muchos complementos desde Complementos de &brandShortName;. Si busca en Internet encontrará muchos otros sitios con complementos.

Cuando pulsa en un enlace para instalar un complemento, &brandShortName; muestra un diálogo que le pide permiso para instalar el complemento. Puede elegir permitir la descarga e instalación o cancelar el proceso.

Usar el administrador de complementos

Los complementos se gestionan en el administrador de complementos. Seleccione Herramientas > Complementos para abrirlo.

Actualizar complementos

Por omisión, &brandShortName; comprobará periodicamente cuándo hay disponibles versiones nuevas de sus complementos instalados. Si hay alguna, le preguntará en el siguiente inicio si quiere instalarlas. Puede cambiar este comportamiento en la pestaña Actualizar del panel Avanzado de &pref.plural;.

Puede buscar actualizaciones manualmente seleccionando el panel Extensiones Temas y pulsando el botón Buscar actualizaciones. Si hay actualizaciones disponibles, el administrador de complementos mostrará un panel Actualizaciones, donde puede elegir las actualizaciones que quiere instalar. Pulse el botón Instalar actualizaciones para actualizar esos complementos. Necesita reiniciar &brandShortName; para que los cambios tengan efecto.

Desactivar, activar y desinstalar complementos

Si quiere deshacerse de una extensión, puede desactivarla temporalmente, para que esté disponible si quiere volver a activarla en el futuro, o bien desinstalarla. Los temas sólo pueden ser desinstalados, ya que todos los temas, excepto el seleccionado actualmente, están desactivados automáticamente.

Para desactivar o activar una extensión, seleccione la extensión que desee y pulse su botón Desactivar o Activar. Para eliminar un complemento de &brandShortName; seleccione el complemento que desee eliminar y pulse su botón Desinstalar. Necesita reiniciar &brandShortName; para que los cambios tengan efecto.

Otras funcionalidades

Otras funcionalidades tales como Visitar página de inicio y Acerca de están disponibles seleccionando el complemento que desee, pulsando &ctrlKey; y haciendo clichaciendo clic con el botón derecho en el complemento y seleccionando el elemento respectivo del menú contextual.

Tareas específicas de extensiones o temas

Configurar extensiones

Las extensiones a menudo proporcionan un diálogo para configurar su funcionalidad. Para abrir el diálogo de &pref.plural; de una extensión, seleccione el panel Extensiones seguido de la extensión que desee y pulse su botón &pref.pluralCaps;, o simplemente haga doble clic en la extensión.

Cambiar el tema

Para alternar entre sus temas seleccionados seleccione el panel Temas seguido del tema que desee y pulse su botón Usar tema. Necesita reiniciar &brandShortName; para que los cambios tengan efecto.

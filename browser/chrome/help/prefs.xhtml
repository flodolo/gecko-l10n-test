一般&pref.pluralCaps;

首頁

這裡可設定做為啟動 &brandShortName; 時或按下首頁按鈕時顯示的網頁或分頁群組。 輸入網址即可。

若按下 使用目前網頁 則會把目前正在瀏覽的所有分頁設為首頁，也可以設定 多個首頁。 當您開啟多個分頁時，此按鈕會將所有分頁群組設為首頁。

若想使用書籤裡記錄的網址，請按使用書籤...。 若選書籤目錄則會用多個分頁同時載入目錄內所有的網址。

若您不想載入任何網頁當首頁，請選使用空白頁。

預設瀏覽器

若您希望 &brandShortName; 永遠都是預設瀏覽器可勾選此項。 另外也可按立刻檢查直接檢查。

連線設定

您的網路環境可能要使用 Proxy 代理伺服器才能正確連線。 這方面的設定請洽詢您的系統管理者或網路提供者 (ISP)。 Proxy 扮演電腦和網路之間中繼者的角色，他會攔截所有的網路請求並 看看能否能從快取中提供資料。Proxy 常用來增加網路速度、過濾資料 和隱藏 IP 增加安全性。企業中的防火牆常需要使用 Proxy。

連線設定視窗:

直接連線到網際網路
這是預設的&pref.singular;，用於不需設定 Proxy 時。
自動偵測此網路的 Proxy 設定
若您希望 &brandShortName; 自動偵測 Proxy 設定 (要所在網路有支援才行)請選取此項。
手動設定 Proxy
若沒有自動設定網址 (PAC) 請向系統管理員或網路提供者(ISP) 詢問伺服器的主機名稱或 IP 與 Port 的資料後填於此處。
Proxy 自動設定位址(URL):
若您的所在網路有 Proxy 自動設定檔，請向系統管理員或網路提供者(ISP) 取得設定檔網址後填入此處後按重新載入。

個人隱私&pref.pluralCaps;

個人隱私設定包含了您曾去過哪些網站、看過哪些網頁、密碼等等。 各項都有清除可用來移除已存的記錄。如果您想刪除所有密碼， 選密碼管理員視窗中的檢視已儲存密碼然後按全部移除鈕。

清除隱私資料

&brandShortName; 可用 &accelKey;+&shiftKey;+Del 或選取 工具 > 清除隱私資料... 以快速清除您使用 &brandShortName; 所留下的種種記錄。

瀏覽記錄

這裡可以指定要瀏覽記錄要保留多久。 預設是 9 天。

已存表單

儲存我輸入於表單與文字搜尋列的內容
若選取此項則 &brandShortName; 會把您輸入的內容全都記錄 並在您填入類似資料時自動給予建議。

密碼

記住密碼
&brandShortName; 可以安全的儲存各項密碼以方便使用網路服務。 若不選此項則 &brandShortName; 就不會儲存密碼。

設定主控密碼...
&brandShortName; 可以用「主控密碼」來保護其它私密的資料如各網站的帳號密碼與憑證等。 建立主控密碼後每次使用 &brandShortName; 且用到被保護的資料時都要先輸入一次主控密碼。

移除主控密碼...
按下此按鈕以移除主控密碼。 這個功能要輸入目前已設的主控密碼才行。

檢視已存密碼
您可以用檢視已存密碼按鈕 以管理、刪除已存的各項密碼。

選取要清除下載記錄的時機: 是 成功下載後、 關閉 &brandShortName; 時或是 手動。 按下立刻清除下載記錄或是下載管理員的 清除可清除已完成檔案的記錄。

Cookies

Cookie 是網站存於您電腦上的資料，用於儲存您的習慣或其它資訊。

允許網站設定 Cookie
此項預設是開啟的。 若關閉則有些網站可能會無法正常運作。

若要顯示Cookie 管理員請按檢視 Cookies。

關於 Cookie 的其它說明請見 管理 Cookies。

快取

瀏覽的網頁通常會在您的電腦上的特殊目錄 (快取) 中有份複本 以加速瀏覽網頁所需的連線與讀取時間。

內容&pref.pluralCaps;

拒絕彈出型(Popup)廣告視窗
&brandShortName; 預設開啟內建擋廣告功能。 關閉此項設定等於停用擋廣告。

有些網站把重要功能如登入做在彈出型視窗裡。 這種情形請選允許的網站並輸入網站再按允許。

若想由允許的列表內刪除某個網站只要選取它後按移除網站即可。 清空列表請按移除全部網站。

允許網站安裝軟體
&brandShortName; 預設允許網站安裝擴充套件與佈景主題等各種軟體 (不過一定會先要求您確認要安裝) 。

此項設定開啟時就可以直接安裝佈景主題。 不過由於擴充套件的功能及危險性大得多， 安裝時您還要另外指定是否信任該網站 (按允許的網站、輸入網址、 並按允許) 。

載入圖片
預設會自動載入網頁所需的圖片。 若不選此項則完全不載入圖片。

  • 只允許同網站的圖片(站內鏈結)
    此選項會停止跨站的圖片資源 (常見的有廣告或計數器) ，只有同網站的內容才可載入。

另外例外可用來設定某些特別想允許或拒絕的網站。

可使用 Java
Java 是由 Sun Microsystems 公司開發給網頁使用的程式語言。 單一個 Java 程式可以在許多不同平台上執行，所以程式設計師就 不需要為不同平台編寫個別的版本就能夠運作。 若要使用 Java，除了開啟此項外還要裝好 Java Plugin。

可使用 JavaScript
JavaScript 是常用於網頁的程序式描述語言(Scripting)， 但與 Java 不同。 JavaScript 常用於動態顯示的網頁。 停用 JavaScript 很可能會讓大多數的網站出問題。

允許網頁自行指定字型

鏈結顏色
改變網頁鏈結的顏色。

鏈結加底線
網頁鏈結預設會加上底線。

允許網頁自行指定色彩
&brandShortName; 預設會照網頁指定的色彩來顯示。 若不選取此項則會將所有網頁都用前面設定的色彩來顯示而忽略網頁的指定色彩。

分頁&pref.pluralCaps;

開啟指定要開新視窗的鏈結時使用:
&pref.singular; 可以設定當網頁指定要開新視窗時，要用新視窗或最後顯示的分頁來開啟網頁。

註:如果您選擇開在新分頁，但網頁設計師已指定要以指定大小的新視窗開啟， &brandShortName; 會忽略這個 &pref.singular; 並開啟新視窗。因為某些頁面只能在特定大小上正常運作。

只有一個分頁時自動隱藏分頁工具列
若 &brandShortName; 視窗內只顯示一個網頁(分頁)，預設不會顯示分頁工具列。

用分頁開啟鏈結後自動切換至新分頁
在鏈結上按中鍵或 &accelKey;+左鍵 時會用新分頁開啟鏈結，且不會直接顯示該分頁 (於背景載入) 。 若想直接切過去 (前景載入) 請選取此項。

關閉多個分頁時發出警告
若關閉視窗時仍有多個分頁開啟， &brandShortName; 可以提醒您是否真的要全部關閉。

這些設定可控制 &brandShortName; 如何處理不同類型的檔案如 應用程式、壓縮檔、或多媒體影音檔。

所有檔案都存入此目錄:
&brandShortName; 預設會設定至桌面以方便您使用下載的檔案。 若要設定為別的目錄請按瀏覽。

下載管理員

開始下載時自動顯示下載管理員
若選取此項則開始下載檔案時會自動顯示下載管理員。

當所有下載完成時關閉下載管理員
選擇要不要在所有檔案下載完成時關閉下載管理員。

全部下載完成後關閉下載管理員
若選取此項則所有下載都結束時會自動關閉下載管理員。

下載處理方式

選取檢視與編輯處理方式可以檢視所有您已設定的檔案類型。 點選個別項目可設定您要 &brandShortName; 如何處理此類檔案。

  • 用預設程式開啟:
    由您的系統設定決定要用什麼開啟此類檔案。

  • 使用此 Plugin:
    若是多媒體影音檔通常會設為此項，用對應的 Plugin 來直接顯示與開啟。

進階&pref.pluralCaps;

這些設定是一般不需要不建議新手自行設定的項目， 但對有經驗的使用者來說可能是非常強大好用的功能。

一般

輕鬆使用界面(Accessibility)

可用鍵盤選取文字
又名鍵盤瀏覽(Caret Browsing)，會多顯示一個代表鍵盤的游標， 可用方向鍵等一般鍵盤操作移動與選取文字等等。 切換此模式的熱鍵是F7。

使用隨打即找(Find as you type)
此項設定開啟時只要您一打字瀏覽器就自動進入搜尋文字模式， 對於快速查詢大量資料非常有用。 但若您因此無法在表單內打字請先關閉此項。

瀏覽

自動縮放圖片以配合瀏覽器視窗大小
選取此項會縮放比螢幕大的圖片，讓它可以完整顯示於您的螢幕上。 想回到原始比例只要在圖片上點一下即可。

使用自動捲動(Autoscrolling)
自動捲動讓您可在壓住滑鼠中鍵時捲動畫面 (若您的滑鼠有滾輪則可由 &pref.singular; 取消)。

使用平滑捲動(Smooth Scrolling)
平滑捲動在您閱讀一堆超長網頁時很有幫助。 通常按下 Page Down 翻頁時會直接顯示下一頁， 開啟此項時有捲頁的動畫效果，讓您可以知道到底捲動了多少行。

語言

語言設定視窗:

更新

&brandShortName; 可以自動檢查是否有佈景主題、擴充套件、或是 &brandShortName; 本身的新版或更新檔可供安裝。

自動檢查是否有更新:
&brandShortName; 預設會自動檢查主程式本身與所有已裝的佈景主題及擴充套件是否有新版。 若不想自動檢查請取消此項。 另外若想手動檢查可按對應的 立刻檢查... 按鈕。

找到 &brandShortName; 的更新檔時,

顯示更新歷史

安全

通訊協定

使用 SSL 2.0
指定使用 SSL2 (Secure Socket Layer Level 2, 標準的安全傳輸協定) 做為檔案傳輸加解密的方法。 所有的正常安全網站都應該支援這種通訊協定。

使用 SSL 3.0
指定使用 SSL3 (Secure Socket Layer Level 3, SSL2 的加強版) 做為檔案傳輸加解密的方法。 但注意可能有網站不支援此種方法。

使用 TLS 1.0
指定使用 TLS (Transport Layer Security, 類似 SSL3 的一種標準) 做為檔案傳輸加解密的方法。 但注意可能有網站不支援此種方法。

憑證

憑證有助於與 安全網站 間連線的加密與解密。 對於下列專有名詞的深入說明請見 政府機關公開金鑰基礎建設 或 亞洲公開金鑰基礎建設論壇。

當網站需要憑證時:
當網站要求建立安全連線時， &brandShortName; 會自動使用合適的憑證。 若想手動選擇憑證請用 &pref.singular; 每次都詢問我。

檢視憑證
按此檢視已安裝的憑證、匯入新憑證、備份或刪除舊憑證。

廢止清冊
&brandShortName; 支援憑證廢止清冊(Certificate Revocation List, CRL) 以確定您的憑證是有效的。 點選廢止清冊按鈕以管理已安裝的廢止清冊。

驗證
OCSP (線上憑證狀態通訊協定，Online Certificate Status Protocol) 是用來驗證您的憑證是否有效，預設不使用。 若想啟用請按此。

安全裝置
安全裝置可用來加密與解密連線並儲存憑證與密碼。 如果您需要使用外接的安全裝置，請點選安全裝置。

防詐騙

瀏覽時檢查是否為詐騙網站
若選取此項，&brandShortName; 會檢查正瀏覽的網站是否為偽造網站，試圖誤導您 送出個人隱私資料。

使用本機儲存的清單
選取此選項後，會使用儲存在電腦中的黑名單來與正瀏覽網站比對，不會和其他網站交換資料。 但詐騙技術日新月異，這種方式很可能不如與其他網站比對來的有效。

與網站比對
此選項會將目前網頁的瀏覽資訊以加密方式送到您所選的網站以比對是否為偽造網頁。 這種方式最有效，但選用前請仔細閱讀該網站的隱私政策。

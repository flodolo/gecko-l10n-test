&pref.pluralCaps;, panel Principal

Inicio

Cuando abre &brandShortName;, se muestra por omisión su página de inicio. Su página de inicio puede ser un sitio web visitado habitualmente, o quizás el acceso a su cuenta de correo; la decisión es suya. Puede cambiar su página de inicio o escoger mostrar una página o conjunto de páginas diferente en esta sección.

Cuando se inicie &brandShortName;
Por defecto, cuando &brandShortName; se inicia abre su página de inicio (escrita en la caja de texto que hay inmediatamente debajo), la cual corresponde a la &pref.singular; Mostrar mi página de inicio. Opcionalmente, puede elegir mostrar una página en blanco al iniciar (quizá para eliminar el tiempo requerido para cargar esa página de Internet) seleccionando la &pref.singular; Mostrar una página en blanco.

Por último, puede hacer que &brandShortName; vuelva a abrir las ventanas y pestañas que tenía abiertas la última vez que estuvo en ejecución, restaurando así la situación en que se encontraba la última vez que estuvo navegando por Internet. Ésta puede ser una manera útil de guardar lo que estaba haciendo cuando cerró &brandShortName;, quizá para instalar una actualización del sistema operativo. Puede elegir esta opción seleccionando Mostrar mis ventanas y pestañas de la última sesión.

Página de inicio
Si ha configurado &brandShortName; para mostrar su página de inicio cuando se inicia &brandShortName;, puede especificar su página de inicio escribiéndola aquí. También puede pulsar cualquiera de los botones bajo la caja de texto para elegir una página de inicio especial:

  • Usar página actual establece como página de inicio la página (o páginas, si tiene múltiples pestañas abiertas) que está visitando actualmente.
  • Usar marcador establece como página de inicio el marcador seleccionado (o si selecciona una carpeta, todos lo que contenga ésta).
  • Restaurar al valor por omisión restablece su página de inicio a la página de inicio por defecto.

Descargas

Esta sección contiene las &pref.plural; que determinan cómo maneja los archivos &brandShortName;.

Con esta &pref.singular; seleccionada, &brandShortName; abrirá la ventana de descargas cuando comience a descargar un archivo.

Cerrarla cuando finalicen todas las descargas
Seleccione esta &pref.singular; para que &brandShortName; cierre la ventana de descargas cuando todas las descargas en curso concluyan.

Guardar archivos en
Por defecto, los archivos descargados se guardan automáticamente en el Escritorio, eliminando la molestia de tener que especificar una ubicación para cada archivo que se descarga. Puede elegir una carpeta diferente pulsando el botón Examinar.

Con esta &pref.singular; seleccionada, &brandShortName; le preguntará dónde quiere guardar cada archivo que descargue (en lugar de usar la ubicación por omisión especificada arriba).

Seleccione esta opción si quiere que &brandShortName; compruebe si es el navegador por defecto al iniciarse. Esto asegura que &brandShortName; se usará siempre que una aplicación intente mostrar una página web. También puede pulsar el botón Comprobar ahora para verificarlo en el momento.

Su organización o proveedor de servicios de Internet puede ofrecer o requerir el uso de un proxy. Un proxy actúa como un intermediario entre su ordenador e Internet. Intercepta todas las peticiones a Internet para ver si puede cubrir la petición usando su caché. Los proxies se usan para mejorar el rendimiento, filtrar peticiones, y ocultar su ordenador de Internet para mejorar la seguridad. Los proxies a menudo son parte de un cortafuegos corporativo.

Descargar e instalar automáticamente la actualización
Cuando esta &pref.singular; está seleccionada, las actualizaciones para &brandShortName; se descargan automáticamente y se instalan la siguiente vez que se reinicie &brandShortName;. La &pref.singular; Advertirme si esto desactivará alguno de mis complementos determina si se le avisará antes de la instalación de una actualización que requeriría que un complemento incompatible con ella fuera deshabilitado porque no existe una versión compatible. Pestaña Cifrado

Protocolos

Usar SSL 3.0
Especifica si quiere enviar y recibir información segura mediante SSL3 (Secure Sockets Layer Level 3, Capa de Conexiones Seguras Nivel 3), un protocolo estándar para comunicarse de manera segura con los sitios web. Desactivar esta opción puede impedirle visitar algunos sitios.

Usar TLS 1.0
Especifica si quiere enviar y recibir información segura mediante TLS (Transport Layer Security, Seguridad en la Capa de Transporte), un estándar de seguridad similar a SSL3. Desactivar esta opción puede impedirle visitar algunos sitios.

Certificados

Los certificados ayudan a realizar el cifrado y descifrado de las conexiones a sitios seguros.

Cuando un sitio web requiera un certificado:
Cuando un sitio web solicita una conexión segura, &brandShortName; usará de modo predeterminado un certificado apropiado. Si desea escoger manualmente un certificado (por ejemplo, si desea usar un cierto tipo de cifrado en lugar del seleccionado automático), escoja la &pref.singular; Preguntarme cada vez y tendrá el control total de qué certificados se usan durante la navegación.

Ver certificados
Pulse este botón para ver los certificados guardados, importar nuevos certificados y hacer copias de respaldo o borrar certificados antiguos en &brandShortName;.

Listas de revocación
&brandShortName; puede usar listas de revocación de certificados (también conocidas como CRLs, Certificate Revocation Lists) para asegurar que sus certificados son válidos. Pulse el botón Listas de revocación para administrar las CRLs instaladas en su ordenador.

Verificación
&brandShortName; puede verificar la validez de sus certificados usando OCSP (Online Certificate Status Protocol, Protocolo de Estado del Certificado en Línea) cada vez que se usan o visualizan. &brandShortName; no usa OCSP por defecto, pero si quiere activarlo, puede hacerlo aquí. Probablemente necesite cambiar esta configuración únicamente si su entorno de Internet lo requiere.

Dispositivos de seguridad
Los dispositivos de seguridad pueden cifrar y descifrar conexiones y guardar los certificados y contraseñas. Si necesita usar un dispositivo de seguridad distinto del de &brandShortName;, pulse el botón Dispositivos de seguridad.

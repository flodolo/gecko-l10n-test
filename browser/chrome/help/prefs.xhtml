使用自动滚屏
自动滚屏是一个很有用的功能，它允许您通过单击鼠标中键（通常是鼠标滚轮）并上下移动鼠标来滚动页面。有些人会觉得这个功能很讨厌，那么可以取消此&pref.singular;禁用自动滚屏。

使用平滑滚动
平滑滚动在您阅读许多很长的页面时会很有用处。 通常，当您按 Page Down，视图会直接跳到下一页。启用平滑滚动，它会平稳的向下滑动，这样您可以看清楚它向下滚动了多大距离。这让您更容易从之前阅读到的地方继续阅读。

输入时即检查拼写
当启用此选项时，&brandShortName; 将在您在网络表格中输入的同时检查您的拼写并提供可能的修正。注意若使用拼写检查您需要下载字典，右键单击按下 &ctrlKey; 并点击任何文本区域，如果需要可以启用拼写检查，然后使用提供的语言菜单下载字典。

语言

有些以多种语言提供的网页。单击选择...按钮可以指定您的首选语言。

语言对话框
要添加一种语言，单击请选择需要添加的语言...，选择一种语言，然后单击添加按钮。在激活的语言列表中选择一种语言并单击移除按钮，可以移除一种语言。您也可以使用上移和下移按钮重新排列语言，指定在页面提供多种语言的情况下优先首选的语言。

网络标签页

连接

您的公司或因特网服务提供商可能提供或要求您使用代理服务器。代理服务器在您的计算机与因特网之间扮演中介的角色，它截取所有向因特网发送的请求，查看它能否使用它的缓存实现这些请求。代理服务器用于改进性能、过滤请求或在因特网上隐藏您的计算机以增强安全性。代理服务器通常是公司防火墙的一部分。

加密标签页

协议

使用 SSL 3.0
指定您是否希望通过 SSL3（Secure Sockets Layer，Level 3）发送和接收安全信息，SSL3 是一种用于与网站安全通信的标准协议。禁用 SSL3 将会阻止您访问某些站点。

使用 TLS 1.0
指定您是否希望通过 TLS（Transport Layer Security）发送和接收安全信息，TLS 是一种类似 SSL3 的安全标准。禁用 TLS 将会阻止您访问某些站点。

证书

证书帮助执行连接到安全站点的连接的加密和解密。

当网站请求证书时:
当一个网站请求安全连接时，&brandShortName; 默认会自动使用适当的证书。如果您希望手动选择证书（例如，如果您希望使用某一类型的加密算法而不是用自动选择的类型），可以选择每次均询问&pref.singular;，这样您将可以完全控制浏览时使用哪种证书。

查看证书
单击此按钮可以查看存储的证书、导入新的证书或者备份或删除 &brandShortName; 中的旧证书。

撤销列表
&brandShortName; 可以使用证书撤销列表（也称作 CRLs）以保证您的证书总是有效。单击撤销列表按钮管理安装在您的计算机中的 CRLs。

验证
&brandShortName; 可以使用 OCSP（Online Certificate Status Protocol）在每次查看或使用证书时验证它们的有效性。 &brandShortName; 默认不使用 OCSP，但如果您希望使用 OCSP，可以在这里启用。很可能只在您的网络环境需要时，您才需要更改这个设置。

安全设备
安全设备可以加密和解密连接以及保存证书和密码。如果您需要使用 &brandShortName; 内置安全设备以外的其他安全设备，单击安全设备按钮。

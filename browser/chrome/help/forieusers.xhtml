สำหรับผู้ใช้ Internet Explorer

หากคุณเคยใช้ไมโครซอฟท์® Internet Explorer คุณก็สามารถเริ่มต้นใช้ &brandFullName; ได้อย่างง่ายๆ

โปรดสังเกต: ข้อมูลเหล่านี้ใช้ได้กับ &brandFullName; สำหรับวินโดวส์และแมคโอเอส

ในส่วนนี้:
  • คำศัพท์ที่ใช้แตกต่างกัน
  • เกี่ยวกับค่าตั้งและรายการโปรดสำหรับ IE ของคุณ
  • คุณสมบัติของ &brandShortName;
  • แป้นลัด

คำศัพท์ที่ใช้แตกต่างกัน

Internet Explorer &brandShortName;
ตัวเลือกอินเทอร์เน็ต &pref.pluralCaps;
แฟ้มอินเทอร์เน็ตชั่วคราว (Temporary Internet Files) แคช
รายการโปรด (Favorites) ที่คั่นหน้าเว็บ
แถบที่อยู่ (Address Bar) แถบที่ตั้ง
ฟื้นฟู (Refresh) เรียกใหม่
แถบเชื่อมโยงแถบรายการโปรด แถบที่คั่นหน้าเว็บ
คัดลอกทางลัด (Copy Shortcut)คัดลอกการเชื่อมโยงไปที่คลิปบอร์ด (Copy Link to Clipboard) คัดลอกที่ตั้งลิงก์
บันทึกเป้าหมายเป็น (Save Target As)ดาวน์โหลดลิงก์ลงดิสก์(Download Link to Disk) บันทึกลิงก์เป็น

เกี่ยวกับค่าตั้งและรายการโปรดสำหรับ IE ของคุณ

ในครั้งแรกของการติดตั้ง &brandShortName; จะนำเข้าค่าตั้งของ Internet Explorer รวมทั้ง รายการโปรด, คุกกี้, รหัสผ่านที่เคยจำไว้ และข้อมูลอื่นๆ ของคุณ ทำให้ช่วยลดเวลาในการปรับแต่ง &brandShortName; ให้เหมาะสมกับความต้องการของคุณ

คุณสมบัติของ &brandShortName;

  • ดูเว็บด้วยแท็บ: แทนที่คุณจะต้องเปิดหน้าต่างใหม่ให้กับทุกๆ หน้าเว็บที่คุณเข้าดู คุณสามารถเปิดหลายหน้าเว็บได้ในหน้าต่างเดียวโดยมีแท็บเป็นตัวแบ่ง คุณยังสามารถจัดกลุ่มของแท็บให้เป็นหน้าแรกของคุณได้ ดูข้อมูลเพิ่มเติมได้ที่ ดูเว็บด้วยแท็บ.
  • ตัวควบคุมหน้าต่างป๊อบอัพ: ทำให้คุณสามารถอนุญาตหรือระงับการเปิดหน้าต่างป๊อบอัพ
  • ตัวจัดการคุกกี้: ทำให้คุณกำหนดได้ว่า &brandShortName; จะทำอย่างไรเมื่อยอมรับคุกกี้
  • ตัวจัดการดาวน์โหลด: จัดสิ่งที่คุณดาวน์โหลดเอาไว้ในหน้าต่างเดียวเพื่อลดความยุ่งเหยิง คุณสามารถเข้าถึงสถิติของสิ่งที่ดาวน์โหลดมาได้โดยไม่จำเป็นต้องเปิดหลายหน้าต่าง
  • ตัวจัดการรูป: อนุญาตให้คุณระงับการแสดงผลรูปจากเว็บไซต์ที่กำหนด หรือจากทุกเว็บไซต์ คุณสมบัตินี้มีประโยชน์ในกรณีที่คุณต้องการลดระยะเวลาในการโหลดเว็บไซต์

ปุ่มลัด

คุณจะสังเกตได้ว่า ไมโครซอฟท์ Internet Explorer กับ &brandShortName; มีหลายปุ่มลัดที่ใช้งานเหมือนกัน ถ้าต้องการดูรายการปุ่มลัดทั้งหมด สามารถดูได้จาก รายการปุ่มลัดของ &brandShortName;.

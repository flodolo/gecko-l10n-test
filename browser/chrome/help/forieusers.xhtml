关于您的 IE 收藏夹和设置

当您第一次安装 &brandShortName; 时，将会导入您现有的 IE 设置， 包括收藏夹、Cookie、已保存的密码、以及其它各种数据。这会为您节省用来设置 &brandShortName; 的时间，让 &brandShortName; 适合您的需要。

&brandShortName; 特色

  • 标签式浏览：您不必为想要访问的每个网站都打开一个单独的浏览窗口，而只需在同一个窗口中通过标签的方式来浏览多个站点。您还可以将一组标签设置为您的主页。想了解更多信息，请查看 标签式浏览。
  • 弹出窗口控制：您可以允许或禁止在前台或后台弹出窗口。
  • Cookie 管理器：允许您更改 &brandShortName; 接收 Cookie 的处理方式。
  • 下载管理器：通过把下载项目保存在一个单独的窗口中，您可以有条理的管理您的下载。您不必打开多个窗口就可以控制所有项目的下载情况。
  • 图像管理器：允许您禁止特定网站载入图像或者禁止载入所有图像，这样有利于减少网页载入时间。

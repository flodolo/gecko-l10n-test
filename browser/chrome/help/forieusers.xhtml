Acerca de su configuración y Favoritos de IE

Cuando instala &brandShortName;, éste importara su configuración actual de Internet Explorer, incluyendo sus Favoritos, cookies, contraseñas guardadas, y otra variedad de datos. Esto le ahorra tiempo a la hora de personalizar &brandShortName; para ajustarlo a sus necesidades.

Características de &brandShortName;

  • Navegación con pestañas: en lugar de abrir una ventana separada del navegador para cada página web se que quiera visitar, se pueden abrir múltiples páginas web en la misma ventana y cambiar entre ellas usando las pestañas. También se puede establecer un grupo de pestañas como página de inicio. Para más información vea Navegación con pestañas.
  • Controles de ventanas emergentes: le permite aceptar o suprimir las ventanas emergentes tanto en primer como en segundo plano.
  • Administrador de cookies: le permite cambiar lo que &brandShortName; hará al recibir cookies.
  • Administrador de descargas: organiza las descargas agrupándolas en una sola ventana para tenerlas más ordenadas. Tiene acceso total a las estadísticas de descarga sin tener que usar múltiples ventanas para cada descarga.
  • Administrador de imágenes: permite desactivar las imágenes de ciertos sitios web o de todos en general. Esto es útil si se quiere disminuir el tiempo que tardan en cargarse las páginas web.

Atajos de teclado

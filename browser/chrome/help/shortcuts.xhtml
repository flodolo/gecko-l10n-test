Atalhos do Teclado do &brandFullName;

Esta é uma lista dos atalhos do teclado mais comuns no &brandFullName;. Se você ativou os atalhos de edição de texto do Emacs no GNOME, eles também funcionarão no &brandShortName;. Quando um atalho de edição de texto do Emacs gerar um conflito com o atalho padrão do &brandShortName; (como acontece com &accelKey;+K), o atalho Emacs terá preferência se o foco estiver em uma caixa de texto (isso inclui o Campo de endereço e o Campo de pesquisa). Nesses casos você deverá utilizar um atalho de teclado alternativo se houver algum listado abaixo.
